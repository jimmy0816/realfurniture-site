## 先搞清楚定義 {#定義}

系統櫃是以工廠預製的板材（多半為塑合板、密集板或發泡板）為基礎，依照現場尺寸組裝而成的收納系統；實木家具則是以原木實材為主要結構，在工坊或工廠加工成型。兩者的核心差異在於「材料」與「生產方式」，衍生出截然不同的特性。

兩者都可以做得好，也都可以做得很差。關鍵在於**你的需求、預算與生活方式**。

## 六大維度完整比較 {#六大比較}

### 1. 價格

**系統櫃不一定比較便宜；實木不一定貴到下不了手。** 台灣市場的現實是，中高端系統櫃報價每尺 NT$4,000–10,000，加上五金、門片、安裝，一間中等主臥衣櫃輕鬆破十萬；而工坊訂製的實木衣櫃，視木種不同，同樣尺寸報價可能落在 NT$8 萬–20 萬。差距存在，但沒有「系統就是便宜」這麼絕對。

| 品項 | 系統櫃 | 實木家具 |
|------|--------|---------|
| 起跳價 | 低（國產板 + 基本五金） | 中（用料決定底線） |
| 上限 | 高（進口板 + 精品五金 + 烤漆門） | 非常高（珍稀木種 + 純手工） |
| CP 值甜蜜點 | 中等預算、大量收納需求 | 重點家具、願意長期投資 |

### 2. 耐用年限

系統櫃的核心材料塑合板（particleboard）對潮濕敏感，台灣氣候中若封邊不確實、螺絲孔反覆鎖脫，十年後結構開始鬆動是常見狀況。一般使用年限約 **10–20 年**，端看板材等級與維護方式。

實木家具若選用適合台灣氣候的木種（如台灣相思木、柚木、橡木）、做好日常保養，**30–50 年以上**不是神話。許多人家裡還留著阿公時代的實木餐桌，換個台腳繼續用。關於木種選擇，可以參考我們的 [木種比較完整指南](/posts/07-wood-species-comparison)。

### 3. 風格彈性

系統櫃勝在「快速落地」。板材顏色、門片款式、把手樣式，廠商 catalog 翻一翻就能決定，設計師也熟悉搭配方式，北歐風、現代簡約、奶油色系都能快速對應。

實木家具的風格天生帶有溫度感，但不代表只能做日式或鄉村風。當代工坊已能做出線條俐落的現代感實木家具，甚至工業混搭風。限制在於**選擇需要更多溝通與時間**，不像系統櫃可以對著型錄下單。

想了解實木家具的客製化空間，可以讀這篇：[客製化實木家具完整攻略](/posts/13-custom-solid-wood-furniture-complete-guide)。

### 4. 搬家友善度

這是系統櫃最大的弱點之一。系統櫃雖然「理論上」可以拆解搬遷，但實際上螺絲孔位二次鎖定後板材容易崩裂，搬家業者也不擅長重組，多數人搬家時只能放棄。

實木家具（尤其獨立件家具）大多可以搬著走，一組實木書桌、實木床架跟著你換三間房子也不成問題。這對租屋族、預期未來會換房的族群，是重要考量。

### 5. 環保性

兩者都有環保議題，但角度不同。

- **系統櫃**：塑合板的黏合劑（甲醛）是主要爭議，低甲醛板材（E1、F☆☆☆☆）已大幅改善，但板材本身難以回收降解。
- **實木家具**：使用的是天然木材，但若非來自永續林場，可能涉及濫伐問題；台灣本地木工坊多使用合法進口材或認證材，相對可追溯。

整體而言，**永續認證的實木 > 低甲醛系統板 > 無認證的低價塑合板**。

### 6. 維修難度

實木家具的修復彈性極高。刮傷打磨補色、榫卯重新補膠、腳架更換，都是工坊可以處理的項目，費用相對合理。我們有一篇 [實木家具刮傷 DIY 修復](/posts/17-solid-wood-scratch-repair-diy) 教你自己處理小損傷。

系統櫃的維修相對麻煩：板材一旦受潮膨脹、封邊剝落，更換零件要配合原廠規格，斷貨時就是大問題。門片鉸鏈五金倒是好換，但結構性損壞往往只能整組換掉。

## 常見迷思破解 {#迷思破解}

### ❌ 迷思一：「系統櫃比較便宜」

**部分正確，但不絕對。** 系統櫃的起跳門檻低，但中高端配置的最終報價並不便宜。特別是加了進口板、烤漆門片、精品鉸鏈之後，一間主臥衣櫃 15–20 萬並不罕見。反而工坊的橡木衣櫃可能只要 8–12 萬。

評估時要做的事：**列出同等尺寸的兩邊報價，再做決定。**

### ❌ 迷思二：「實木不能做衣櫃，只能做餐桌」

**完全錯誤。** 實木衣櫃、實木書櫃、實木鞋櫃，都是工坊長期在做的品項。現代工坊使用集成板、榫卯結構，一樣能做出穩定、實用的收納家具，外觀上也可以做到相當精緻。

限制只有一個：**不適合潮濕環境直接接觸水氣**（如衛浴、廚房備餐台），這類環境反而更適合特殊塗裝處理的板材。

### ❌ 迷思三：「系統櫃可以搬走繼續用」

**理論成立，現實很難。** 板材一旦拆解再組裝，螺絲孔位變大、板材結構弱化，通常撐不了幾次搬遷。真正搬家友善的是**獨立件實木家具**，或是特別設計為「可拆裝」的模組家具。

### ❌ 迷思四：「實木家具在台灣一定變形」

**沒有照顧才會，有照顧就不會。** 台灣高濕氣候確實對實木有要求，但選對木種（含水率穩定）、做好表面塗裝保護、保持通風，實木家具可以在台灣好好活幾十年。詳細保養方式可參考 [實木家具在台灣的照顧指南](/posts/03-solid-wood-care-taiwan-climate)。

## 混搭策略：哪些做系統、哪些用實木 {#混搭策略}

「全系統」或「全實木」都不是最優解。里爾研究室建議的**混搭邏輯**：

### ✅ 適合系統櫃的空間

| 空間 / 品項 | 理由 |
|------------|------|
| 主臥衣櫃（整面牆） | 量體大、尺寸需精準切合空間、收納機能優先 |
| 廚房上下櫃 | 需防潮塗裝、尺寸客製、與家電整合 |
| 更衣室 | 大量收納、分格細、CP 值高 |
| 浴室收納 | 防水板材比實木更合適 |
| 電視牆收納（側邊儲物） | 量體大、預算有限時的解法 |

### ✅ 適合實木家具的空間

| 空間 / 品項 | 理由 |
|------------|------|
| 餐桌、書桌 | 每天接觸、使用頻率高、質感有感 |
| 床架 | 安全性、耐久性、睡眠空間的氛圍主角 |
| 椅子 | 實木椅比系統板更有修復彈性 |
| 書房書架（獨立件） | 搬家帶走、未來換風格可重組 |
| 展示用邊几、茶几 | 視覺焦點、實木紋理質感加分 |

**里爾師傅的口訣：「收納空間系統做，生活家具實木放。」**

| 廚房上下櫃 | 系統廚具 | NT$12–18 萬 |
| 餐桌（6 人） | 實木 | NT$2.5–4.5 萬 |

| 小孩房衣櫃 | 系統 | NT$4–6 萬 |
| **合計** | | **NT$39–63 萬** |

### 版本 B：實木優先（預算 NT$75 萬）

| 品項 | 類型 | 預算 |
|------|------|------|
| 主臥實木衣櫃（訂製） | 實木 | NT$15–25 萬 |
| 廚房上下櫃 | 系統廚具 | NT$12–18 萬 |
| 餐桌 + 椅（整組） | 實木 | NT$5–10 萬 |

| 書桌 + 書架 | 實木 | NT$3.5–7 萬 |

| **合計** | | **NT$47.5–83 萬** |

> **里爾建議**：廚具因為需要整合電器、防潮、標準化尺寸，不論哪個版本都建議走系統廚具。把省下來的錢，投資在你每天最有感的家具上。

## 里爾師傅怎麼說 {#師傅觀點}

我們請教了工坊的黃師傅，他做實木家具超過 25 年，也見過很多客戶在系統和實木之間掙扎。

> 「很多人來問我，是不是所有東西都要做實木才好？我都說，不一定。廚房我自己家也是系統廚具，因為那個環境不適合。但餐桌、床架，我絕對用實木。每天吃飯、每天睡覺，品質差一點你會感覺到。」

> 「有人覺得實木很貴，但我看過很多家庭，系統衣櫃做了十幾年換了三組，每次都要花重新施工的錢；旁邊的實木衣櫃打個蠟繼續用。長期算下來，實木不一定貴。」

> 「最划算的策略？**大面積收納用系統，每天接觸的家具用實木。** 這樣質感和預算都顧到了。」

## 結論：什麼情境選什麼 {#結論}

| 你的情況 | 建議 |
|---------|------|
| 預算有限，收納需求大 | 系統櫃為主，實木買一兩件點睛 |
| 未來有搬家計畫 | 重點家具選實木（可帶走），收納空間租屋不做 |
| 重視生活質感、願意長期投資 | 餐桌、床架、書桌優先實木 |
| 廚房、浴室收納 | 系統/防潮板材，實木不適合 |
| 想要整體風格一致 | 與設計師溝通混搭比例，不必全選一種 |
| 有小孩，使用頻繁 | 實木耐用度更高，長期划算 |

**最後一句話**：系統和實木不是對立的，是互補的。聰明的裝潢，是知道在哪裡花值得的錢。

## 常見問題 FAQ {#faq}

**Q1：系統櫃甲醛問題嚴重嗎？現在還有這個問題嗎？**

現在市面上主流系統板材多已達到 E1 甲醛標準（甲醛釋放量 ≤ 0.1 mg/m³），進口品牌板材甚至達到更嚴格的 F☆☆☆☆。新裝潢完成後保持通風 1–3 個月，甲醛濃度會持續下降。選購時務必要求廠商出示板材甲醛認證文件。

**Q2：整間臥室做實木家具，費用大概是多少？**

以主臥為例（含床架、床頭板、雙門衣櫃、床頭櫃 × 2），使用橡木或白蠟木的工坊報價大約落在 NT$20–50 萬，視尺寸、木種、五金配置而定。可以請工坊分批施作，先做床架，下一年再做衣櫃。

**Q3：系統衣櫃做了之後後悔，可以改嗎？**

大部分情況下，系統衣櫃「整組拆掉重做」是最實際的方案，局部修改空間有限。這也是為什麼我們建議在規劃階段就多花時間討論需求，避免事後改動的高成本。

**Q4：實木衣櫃台灣很潮濕，板材會不會發霉？**

實木衣櫃通常使用集成板（拼接實木）或厚實木板製作，表面有塗裝保護，正常使用情況下不易發霉。建議衣櫃內放除濕棒，保持通風。若是梅雨季節，開除濕機即可。發霉通常是通風不足造成，與材質關係不大。

**Q5：一定要找設計師才能規劃嗎？還是可以直接找工坊？**

可以直接找工坊，特別是你已經知道自己需要什麼的時候。工坊通常會提供現場丈量、3D 平面圖（部分工坊），幫你確認尺寸和功能需求。若空間格局複雜、需要整體動線規劃，找室內設計師整合會更有效率。
