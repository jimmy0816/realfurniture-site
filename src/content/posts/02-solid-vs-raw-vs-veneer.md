## 名詞定義：三種材質的本質差異

### 原木（Raw Wood / Solid Log）

「原木」是最純粹的形式——一整棵樹鋸下來的木材，沒有拼接、沒有加工、沒有填充物，就是一整塊木頭。

你看到的邊桌用一整塊木板做桌面（常見的「一塊木」桌板），或者木頭還保留樹皮邊緣的「活邊桌」，這些就是原木的代表。

原木的特徵：
- 紋路從頭貫穿到尾，每一側都能看到連續的木紋
- 有自然的節疤、色差，每一塊都是獨一無二
- 重量最重（密度高）
- 一定有「木材本身的香氣」

### 實木（Solid Wood / Engineered Solid Wood）

「實木」在台灣的商業語境下，意思比較廣。它包含了：

**全實木（Full Solid Wood）**：整件家具從裡到外都是實心木材，包含桌腳、桌面側面全部是真實木紋。這是最接近「原木」概念的商業用語。

**集成材（Finger-jointed / Glulam）**：把較小的木塊用榫接或膠合方式拼在一起，形成一塊較大的板材。這依然是「實木」的一種——所有木料都是真實木，只是以拼接形式呈現。台灣市場上大量「實木餐桌」屬於這類，CP 值高，耐用度也不差。

**直拼板（Edge-glued Panel）**：同樹種的窄木板橫向拼合，每塊都是真實木，只是拼在一起。常見於書桌桌面。

重點：**集成材和直拼板都算實木，只要核心材料是真實木頭，沒有密集板（MDF）或夾板（Plywood）當心材，就是實木家具。**

### 貼皮（Veneer / Wood Veneer）

貼皮是最容易被誤解的類型。它的結構是：

- 核心：密集板（MDF）、夾板（Plywood）或粒片板（Particleboard）
- 表面：薄薄一層真實木皮（通常 0.3mm - 0.6mm）

外觀上，貼皮家具的表面確實是真實木頭——你看到的木紋是真的，但只有這薄薄一層。底下的「肉」根本不是木頭。

這不代表貼皮家具一定是爛貨。高品質的貼皮工藝能維持相當好的外觀，且貼皮家具因為核心材均勻，比純實木更不容易因為濕度變化而變形。問題在於——**它的壽命天花板比實木低很多，而且磨損或受潮後幾乎無法修復**。

## 製程圖解：從原木到你家桌子，發生了什麼事

想真正理解這三者的差異，最好的方式是跟著木頭的「旅程」走一遍。

**原木路徑：**

```
原始樹幹 → 鋸木廠大片鋸切 → 自然風乾或窯乾（3個月-2年）
→ 依尺寸裁切 → 手工打磨 → 塗裝（推油或上漆）→ 你家桌子
```

這條路最費時費工，但你得到的是一塊跟 50 年前一樣的真實木頭。

**集成材路徑：**

```
原始樹幹 → 鋸成小料 → 窯乾 → 指接膠合 → 壓制成大板
→ 裁切打磨 → 塗裝 → 你家桌子
```

這條路用了更多零碎木料（更環保，減少浪費），成品穩定性比大塊原木更均勻，因為把不同方向的木材紋路交叉結合，減少了單方向收縮膨脹的問題。

**貼皮路徑：**

```
原始樹幹（只用最外層） → 旋切成薄片（0.3-0.6mm）
                              ↓
核心：廢木料 + 膠水 → 壓制成密集板/夾板
                              ↓
薄木皮貼合到核心板 → 上漆/塗裝 → 你家桌子
```

看到了嗎？一棵樹的絕大多數木料變成了「核心」，只有最漂亮的外皮被切成薄片貼表面。這讓貼皮傢俱看起來比成本高出許多。

## 常見誤區：這些話術要注意

### 誤區一：「純實木」= 原木

很多業務說的「純實木」其實是集成材。集成材確實是實心木料，不是貼皮，但它不是一整塊原木。這不代表集成材不好——它在結構穩定性上甚至可能優於原木——但你要知道你買的是什麼。

**問法建議**：直接問「桌面側邊切開來是什麼？有沒有拼接線？」或要求看桌腳的切面。

### 誤區二：重量就代表實木

「這桌子好重，一定是實木」——錯。密集板（MDF）其實非常重，有時比集成材還重。重量是參考，但不是決定性指標。

**正確做法**：同時看重量＋敲擊聲音＋側邊紋路，三者合一才準確。

### 誤區三：「原木色」「木感」代表木質

這是最基礎的陷阱。「原木色」是顏色描述，不是材質描述。貼皮家具、仿木紋貼紙（PVC 貼皮）、甚至塑膠噴漆都可以做出「原木色」外觀。

**特別注意**：仿木紋貼皮（非木皮貼皮）的表面完全沒有木頭，是塑膠薄膜印出木紋，但外觀看起來跟木皮貼皮幾乎一模一樣。辨識方式：用指甲輕刮側邊，如果是塑膠膜，會起白邊；木皮刮起來有纖維感。

### 誤區四：「台灣製造」就代表好

台灣確實有頂尖的木工職人，但「台灣製造」四個字本身不代表材質。台灣也有工廠生產高 CP 值的貼皮傢俱，完全沒問題——問題是你要知道你在買什麼。

### 誤區五：貼皮一定是爛貨

高端設計品牌如 Herman Miller、Fritz Hansen 也使用貼皮工藝——因為薄木皮可以呈現更精準的木紋方向和對稱美感。問題不在材質本身，而在「你以為自己買的是什麼」。

## 選購建議：不同預算怎麼選

### 預算 1.5 萬以下

在這個預算內，你能買到的「實木」選項非常有限。與其追求「全實木」卻買到劣質集成材，不如誠實面對預算，選擇：

- **高品質貼皮家具**：選核心材為夾板（Plywood）而非密集板（MDF）的產品，夾板結構更穩，遇到濕氣不容易膨脹。
- 避免純白色或高亮面貼皮——這種規格最容易磨損後露底。
- 查看側邊或底部，確認木皮厚度（問店家 0.3mm 還是 0.6mm，越厚越好）。

### 預算 1.5-5 萬

這是集成材全實木的主戰場。

- 優先選擇有含水率控管的品牌（問窯乾認證、問進貨產地）
- 要求看桌面側邊的木材截面，確認沒有密集板
- 這個預算買白橡木集成材可以買到非常不錯的品質，用個 20-30 年沒問題
- 避免這個價格帶標榜「胡桃木」或「柚木」的全實木——通常是噱頭

### 預算 5-12 萬

進入全實木胡桃木、白橡木、相思木的正確選購區間。

- 這個價格帶可以要求客製尺寸（因為台南多數職人工坊接受訂製）
- 詢問工法：榫卯結構還是螺絲？有沒有伸縮縫設計（讓木頭有空間呼吸）？
- 直接到工坊看料，這才是台南買家的優勢

### 預算 12 萬以上

原木大板、珍稀樹種的選購區間。

- 「一塊木」的活邊桌在這個價格帶才是真品
- 要求出具木材來源證明（合法砍伐或回收老木料）
- 找有口碑的工坊，看他過去作品，問能否提供使用保固

## 常見問題

**Q1：貼皮家具可以買嗎？還是應該直接選實木？**

A：可以買，但要看使用情境。如果是不常接觸水氣的書房書桌，或者是 5-10 年後會更換的家具，高品質貼皮（核心是夾板）完全夠用。如果是餐桌、廚房邊桌這種長期接觸水分的地方，一定要選集成材以上的實木。

**Q2：集成材算不算真正的實木？**

A：算。集成材的每一塊木料都是真實的木頭，只是用膠合拼接的方式組成更大的板材。它不是貼皮，側面切開你看到的是連續的木纖維，不是密集板的細碎木屑。現代窯乾處理的集成材，耐用度和尺寸穩定性其實優於很多原木。

**Q3：原木大板真的值得那個價格嗎？**

A：這個問題取決於你怎麼定義「值得」。如果你把家具當做投資或傳家之物，原木大板的物質稀缺性和獨一無二感確實難以替代。如果你只是需要一張吃飯的桌子，集成材能提供 80% 的美感和 100% 的功能，用三分之一的價格。

**Q4：怎麼知道業務說的「實木」是哪一種？**

A：直接問三個問題：（1）桌面側邊切開來是什麼？（2）有拼接嗎？（3）有窯乾處理嗎？如果業務答不出來，或者答案吞吞吐吐，通常就不是你期待的那種「實木」。

**Q5：原木和實木哪個更環保？**

A：集成材其實比原木大板更環保——它利用了更多木料（包括邊角料），減少浪費。原木大板需要砍伐更大的樹木，資源消耗更高。貼皮家具的核心材料中含有大量化學黏著劑，甲醛釋放量是三者中最高的，環保性最差。
