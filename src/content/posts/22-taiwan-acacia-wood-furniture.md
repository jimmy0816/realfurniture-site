去朋友家吃飯，看到餐桌上一片深紅棕色的木板，紋路像流水，顏色深淺交錯，完全不像進口木。問了才知道：「這是台灣相思木，我阿公留下來的。」

那張桌子撐了超過五十年，上漆三次，桌面紋路還是清清楚楚。

相思木（Acacia confusa）一直是台灣廟宇、礦坑、農具的常備材料，卻很少出現在現代傢俱展示間。不是因為不好，而是大家不熟悉它——這篇文章想讓你認識這個被嚴重低估的本土木材。

## 相思木是什麼？台灣原生的木材故事 {#相思木是什麼}

**學名：** *Acacia confusa* Merr.  
**科名：** 豆科（Fabaceae）含羞草亞科  
**俗名：** 台灣相思、香絲樹、細葉相思樹、黃金樹  
**生長海拔：** 低海拔約 100–600 公尺

相思樹原生於台灣恆春半島，耐強風、抗乾旱，根部常有固氮的根瘤菌共生，能改善貧瘠土壤。日治時期被大規模種植於全台，至今仍是水土保持的重要樹種。

### 相思木與客家文化的淵源

相思木在台灣的歷史不只是木材史，更是庶民生活史。

早期客家人移民台灣，多開墾山坡地帶，相思樹正好適應這樣的環境。它密度高、份量重、紋路特殊，客家人家中多有相思木製成的農具與傢俱；砍下的分枝則燒成薪碳，客家人沿著山路挑炭下山，這些路徑被保留至今，稱為「挑炭古道」。

值得一提的是，相思木曾是台灣北部礦坑的主要支撐材料。礦工們有一句話：「聽到相思木在哭，就快逃。」——因為木材會在斷裂前先發出聲響，成為危險的預警信號。這說明相思木在強度上的可靠，連礦工的性命都曾交給它。

## 木材特性：數據說話 {#木材特性}

| 特性 | 數值/描述 |
|------|-----------|
| 氣乾比重 | 0.75–0.95 g/cm³ |
| Janka 硬度 | 約 1,700–1,750 lbf（依品種） |
| 心材顏色 | 紅褐色至深棕色 |
| 邊材顏色 | 黃褐色（與心材對比明顯） |
| 紋理 | 斜行紋理，木肌細膩 |
| 光澤 | 富有天然光澤 |
| 氣味 | 淡淡木質香 |
| 耐腐性 | 優（天然油脂抵抗濕氣與蟲蛀） |
| 加工性 | 中等（硬度高，鋸切需鋒利刀具） |

> **數據來源：** 氣乾比重參考自根源木材（twbwood.com）相思木材資料庫；Janka 硬度參考 k-timbers.com、tier1furnishings.com

### 顏色對比是最大的視覺特徵

相思木心材（中心部分）呈深紅褐色，邊材（外圍部分）是明顯的黃褐色。這種強烈的色差在一塊板面上同時出現，是相思木最辨識度最高的外觀特徵——有人覺得野性美，也有人認為不夠整齊。

若你看過大理石紋路的木桌，那種「每一片都不一樣」的感覺，相思木也有，只是是木頭的版本。

## 相思木做傢俱的優點 {#優點}

### 1. 硬度高、耐磨損

相思木的 Janka 硬度約在 1,700 lbf 上下，比常見的胡桃木（約 1,100 lbf）和白橡木（約 1,360 lbf）更硬。這代表日常使用中，刮傷、壓痕的機率更低。

餐桌每天承受碗盤、筷子、小孩的各種動作，硬度高就是長期耐用的基礎。

### 2. 天然含油脂，抵抗台灣濕氣

台灣平均濕度常態性超過 75%（中央氣象署統計），梅雨季節更高。相思木天然含有油脂，對濕氣、蟲蛀的抵抗力強，比起需要密集保養的進口木種，維護門檻相對低。

### 3. 本土木材，碳足跡低

不需要從北美或東南亞飄洋過海，相思木在台灣本地種植，運輸里程短，符合永續採購的方向。喜歡本土、在地優先概念的買家，這是很有說服力的選擇。

### 4. 紋路獨特，每一件都是唯一

心材的深紅棕和邊材的黃褐，搭配出野性的對比紋路。相思木沒有辦法「量產一致」，每一張桌子、每一把椅子都長相不同，這是機器無法複製的質感。

### 5. 固碳率名列前茅

根據相關研究，相思樹是台灣固碳率第一名的樹種之一，作為木材使用還能鎖住碳，符合現代對環境責任的期待。

## 相思木的缺點與限制 {#缺點}

### 1. 樹幹分支早，大板材難取得

相思樹在樹幹低處就開始分支，這是它最主要的材料限制。大型完整板材（如超過 120 公分寬的整塊餐桌面）非常難找，多數相思木傢俱採用集成拼板設計，或是較小尺寸的單體傢俱。

### 2. 邊材與心材色差大

深紅與淺黃的對比是特色，但如果你偏好顏色均勻的桌面，這可能是視覺上的挑戰。喜歡「一眼看起來一致」的買家，需要提前確認板材組合的方式。

### 3. 硬度高反而容易裂

硬木材質在乾濕交替的環境下，因膨脹收縮率較大，**比軟木更容易出現裂縫**。特別是：
- 冷氣長時間直吹
- 環境濕度驟降（如冬天開暖氣）
- 未充分窯乾的原料

預防方式：避免直吹冷氣，保持室內濕度在 45–65% 之間（可參考 d3wood.com 的建議），並確認購買時詢問板材是否有窯乾處理。

### 4. 知名度低，市面上選擇少

相思木傢俱在市場上的能見度遠不及橡木、胡桃木，部分消費者對它陌生，導致市場規模小、選擇少。這種情況正在慢慢改變，但目前找到專門的相思木傢俱工坊需要多花一些時間。

## 如何辨識真正的相思木 {#辨識方法}

| 辨識方式 | 相思木的表現 |
|---------|------------|
| **看顏色** | 心材深紅棕、邊材黃褐，對比明顯 |
| **看年輪** | 年輪清晰可見，紋路呈斜行排列 |
| **摸手感** | 木肌細膩，打磨後質地柔滑 |
| **看光澤** | 表面有天然油脂反射的光澤感 |
| **聞氣味** | 淡淡木質香，不刺鼻 |
| **問產地** | 台灣本土，低海拔山區 |

要特別注意的是，市面上部分「相思木」製品其實是其他品種的相思屬（Acacia spp.）進口木材，並非台灣原生的 *Acacia confusa*。兩者特性略有不同，購買時可以直接詢問店家木材的具體來源。

## 台灣氣候下的保養要點 {#保養要點}

台灣全年平均濕度高，相思木雖然天然耐腐性好，但仍需要基本照護。

### 日常保養

**每季一次：**
- 用乾布擦拭桌面灰塵
- 視情況補上薄層木蠟油或食用級油（如亞麻仁油），補充表面的保護層

**梅雨季（約 5–6 月）：**
- 室內濕度容易超過 75%，建議使用除濕機將濕度控制在 **65% 以下**
- 避免水分積在桌面，擦掉積水

**冬季 / 長期開冷氣期間：**
- 冷氣直吹是相思木裂縫的主要原因之一，建議調整風向或改變傢俱擺放位置
- 若室內過乾（濕度低於 40%），可放一盆水或使用加濕器

### 如果已經出現裂縫

細微裂縫（小於 1mm）通常是自然呼吸的結果，不影響結構，可用相近色系的木工膩子填補再打磨。若裂縫持續擴大，建議聯絡製作工坊評估。

## 選購建議

相思木傢俱適合以下需求的買家：

✅ **預算有限，但想要真實木的質感**  
✅ **偏好獨特、不重複的紋路**  
✅ **注重本土、永續概念**  
✅ **需要高耐用性的工作桌、餐桌**  
✅ **對每件都是「獨一無二」有認同感**

選購時建議確認：
1. 板材是否有窯乾處理？（含水率應在 8–12%）
2. 心材比例是否足夠？（心材耐腐性遠高於邊材）
3. 拼板的方式和膠合品質如何？

## 小結：被低估的台灣驕傲

相思木不是傢俱業的主流明星，它沒有胡桃木的時尚感，也沒有柚木的國際光環。但它是台灣山林裡生長了幾百年的實力派——硬度足、耐腐佳、紋路獨特，還有一段與礦工、客家人共同走過的島嶼歷史。

如果你在選一張陪伴家庭數十年的桌子，相思木值得被認真考慮。

## 常見問題

**Q1：相思木家具會不會很快就裂開？**  
A：相思木硬度高，在濕度變化劇烈的環境下確實比軟木更容易出現裂紋。關鍵在於板材的含水率處理（窯乾）和日後保持室內濕度穩定（建議 45–65%）。購買前詢問窯乾狀況，日常避免冷氣直吹，可以有效降低裂縫風險。

**Q2：相思木和進口 Acacia 集成木哪個比較好？**  
A：台灣相思木（*Acacia confusa*）和進口相思屬木材是不同物種，特性略有差異。台灣相思木氣乾比重 0.75–0.95，密度偏高，耐腐性佳；進口 Acacia 集成木（常見為東南亞品種）品質依品種和加工方式不同。選購時建議詢問具體產地與物種名稱。

**Q3：相思木桌面可以用濕布擦嗎？**  
A：可以，但不要讓水分長期積在桌面。日常清潔用半乾的布輕擦即可，擦完用乾布吸水，避免木材吸水變色或起毛刺。有木蠟油或油脂塗裝的桌面耐水性更好。

**Q4：相思木可以做戶外傢俱嗎？**  
A：相思木的天然耐腐性適合部分戶外使用，但如果長期暴露在直射日光和雨水下，仍需要定期塗裝維護。與柚木或防腐處理木材相比，耐候性略遜，建議評估環境條件後再決定。

**Q5：台灣哪裡可以買到相思木傢俱？**  
A：目前市面上專門以相思木為主材的傢俱工坊較少，大多需要客製化訂製。部分傳統木工坊或本土木材品牌有提供，也可以向傢俱工坊詢問是否能以相思木料製作你想要的款式。有任何問題也歡迎私訊里爾家具研究室，我們可以協助你找到合適的方向。
