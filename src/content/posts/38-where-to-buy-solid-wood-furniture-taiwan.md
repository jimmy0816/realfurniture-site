## 路線一：線上購買 {#線上購買}

### 平台類型

台灣實木家具的線上購買大致分兩類：

**① 綜合購物平台（蝦皮、momo、PChome）**

這類平台的優勢是比價方便、購物保障明確，信用卡折扣和平台優惠活動時能撿到不錯的價格。實木家具在上面的賣家品質不一，有品牌官方旗艦店，也有小廠商或代理商。

挑選注意事項：
- 優先選有「品牌旗艦店」認證的店鋪
- 看評價時要過濾「近 3 個月」而非整體，避免品質下滑看不出來
- 問清楚木材是集成材、全實木還是貼皮，賣家敢清楚說明通常比較可信

**② 品牌自有官網（含 LINE 直購）**

部分台灣實木品牌有自建官網或透過 LINE 官方帳號直購，省去平台抽成後有時價格會比平台便宜。這類通路也較容易問到木材規格、含水率說明。

### 線上購買的三大風險

**風險 1：無法確認木材厚度與重量**

照片可以修，但木頭的厚度和分量感是騙不了人的。同樣標榜「實木餐桌」，4 公分厚板和 2.5 公分厚板差在重量和扎實感——螢幕看不出來。**建議：問賣家桌板厚度（單位：公分）和整桌重量（公斤），若賣家答不出來就要小心。**

**風險 2：色差問題**

螢幕色溫和實際木頭色差有時相當大，特別是胡桃木和橡木這類紋路豐富的木種，天然變化很大。建議在下單前請賣家提供實品照（非渲染圖或棚拍圖）。

**風險 3：退換貨困難**

大型家具的退換運費動輒數千元，消費者保護上常有糾紛。建議選擇「到貨簽收前可檢查」或提供「14 天鑑賞期不計退貨運費」的品牌。

### 什麼人適合線上購買？

- 已在門市或展間看過同款、只是決定在網路下單取得更好折扣
- 預算有限，願意接受集成材或固定規格款
- 已有使用實木家具的經驗，知道自己要什麼

## 路線二：品牌門市 {#品牌門市}

### 台灣主要實木家具品牌門市

**詩肯柚木（Scanteak）**
台灣知名度最高的柚木傢俱連鎖品牌，門市分布全台各大城市。以北歐丹麥為設計靈感，主打柚木材質，耐用性強、適合台灣氣候。售後服務體系完整，適合第一次購買實木家具、想要「買了安心、不用傷腦筋」的族群。價格屬中高端，但有全台服務網絡。

**ALOT Living**
進口日本設計、在台灣銷售的實木家具品牌，風格偏日式現代簡約，選材著重環保無毒。適合喜歡北歐日式混搭（Japandi）風格的買家。

**DAaZ 自然人文**
設計感強，受 Mobile01 和 PTT 網友討論度高，家具風格介於日系與台灣在地之間，常見於中高端裝潢規劃中。

**優渥設計**
強調整體空間設計，家具多半搭配空間規劃一起賣，適合有裝潢需求、想同時搞定設計與傢俱的買家。

### 門市購買的優勢

1. **能摸到實物**：確認木紋質感、表面塗裝、接縫工法
2. **現場比較多款**：同個空間看不同木種的色澤差異
3. **業務可諮詢**：對木材不熟的新手，有人可以問
4. **完善售後**：品牌通常有固定保固期、到府維修服務
5. **分期付款**：多數門市支援信用卡分期

### 門市購買的缺點

- **價格較高**：含陳列成本、人事費用，通常比工廠直營貴 20-40%
- **款式固定**：多半無法客製尺寸，或客製費用另計
- **業務壓力**：部分門市銷售積極，看展時容易被帶節奏

### 逛門市前的準備

去門市之前先做這幾件事，可以大幅提升效率：

1. **量好你的空間**：餐廳/臥室/書房的尺寸，含動線（走道至少留 70-80 公分）
2. **確定木種偏好**：看網路圖片，大致鎖定淺色（橡木、梣木）或深色（胡桃木、柚木）
3. **設定預算上限**：進門市前心裡要有數，不然業務一帶就容易超標
4. **帶空間照片**：讓業務更快理解你的需求

## 路線三：工廠直營 {#工廠直營}

「工廠直營」在台灣的定義比較寬，大致分兩類：

### 類型 A：MIT 生產工廠直銷

從木材進口到家具製作全程自營，省去中間商利潤，價格通常是三條路中最實惠的。

**有木 UMU（台北吳興街）**
台北知名的原木家具工廠直營店，主打手工推油塗裝，強調「從挑選木材就開始把關」。提供客製尺寸，餐桌、書桌、工作桌、中島桌等都做，在 Mobile01 和 PTT 討論度高。

**高仕隆原木傢俱**
從木材進口到家具製作一條龍自營生產，強調 MIT 品質，提供永久保固。適合重視長期使用保障的買家。

**台中地區工廠**
台中有時被稱為「台灣家具之都」，聚集不少實木家具工廠，如瑪荷尼家具、佳昕家居等，價格相對台北門市實惠。如果你在中部、或願意跑一趟，是很值得的選項。

### 類型 B：小型木作工作室（直人品牌）

台灣近年出現不少以設計師或職人為核心的木作品牌，透過 Instagram、Facebook 或個人官網接單，家具款式少但質感強，多半全客製化。

這類工作室通常：
- 接受完全客製（尺寸、木種、塗裝自選）
- 交期較長（8-16 週）
- 價格比大品牌門市低，但比大量生產的工廠高
- 溝通成本高，需要花時間確認細節

**怎麼找這類工作室？** 在 Instagram 搜尋「#木作訂製」「#實木家具台灣」「#手工家具」，通常能找到有在接單的在地職人。

### 工廠直營或木作工作室的注意事項

1. **先確認有無實體展示**：部分工作室沒有陳列空間，只能看照片，木紋色澤以實品為準
2. **確認估價方式**：是「木料重量計費」還是「工時計費」，不同算法影響最終總價
3. **索取含水率說明**：請問是否有做窯乾處理，台灣室內建議含水率 8-12%
4. **確認收尾工法**：是推油、上蠟油還是噴漆，不同塗裝對日常保養需求差很多
5. **溝通交期與定金比例**：通常收 30-50% 訂金開始製作，交貨驗收後付尾款

## 混合策略：先網路做功課，再實體定案 {#混合策略}

如果你不確定要走哪條路，試試這個流程：

```
Step 1：網路研究（1-2 週）
→ 確定木種偏好（淺色 / 深色）
→ 確定風格方向（日式 / 北歐 / 台式）
→ 確定大致預算

Step 2：門市親訪（1-2 次）
→ 摸到實物，確認紋路質感
→ 確認尺寸是否符合空間
→ 詢問木種、含水率、塗裝工法

Step 3：決定通路（最終下單）
→ 款式確定 + 有門市 → 直接在門市或官網下單
→ 有明確客製需求 → 詢問工廠直營或木作工作室
→ 預算有限、已看過實物 → 回到網購平台比價
```

這條路線最省時間，也最少遺憾。

## 結語

實木家具不是消耗品，選對了可以用幾十年。「去哪買」這件事，說穿了就是**在你的時間、預算和需求之間找平衡**：

- 有時間、想看實物 → 門市或工廠
- 已確定款式、想省錢 → 網購比價
- 有特殊尺寸需求 → 直接找工廠或職人

不管你在台灣哪個城市，現在聯繫實木品牌或詢問木作工作室都很方便，不一定要住在門市附近才能買到好家具。

## 常見問題 {#常見問題}

**Q1：網路上買的實木家具可靠嗎？**

可靠，但要挑通路。在蝦皮、momo 上有品牌旗艦店的通常比較有保障，可以看評價、確認退換貨政策。最怕的是「圖很美但賣家說不清楚木材來源」這種，遇到就要多問、多確認。

**Q2：工廠直營一定比門市便宜嗎？**

通常是，因為省了展間成本和中間商利潤，但不是絕對。部分工廠直營的客製家具，因為少量生產、手工比例高，最終報價不一定低於大型品牌門市。重點是：工廠直營的優勢是「彈性」，不只是價格。

**Q3：客製化實木家具怎麼確認品質？**

下單前請工廠或工作室提供：① 木材含水率數據、② 以往作品的實品照（非渲染圖）、③ 使用的塗裝品牌（如 Osmo 木蠟油）。這三樣能問清楚的，通常是可信賴的供應商。

**Q4：台灣各地買家具有地域限制嗎？**

現在幾乎沒有。多數品牌和工廠都提供全台配送，部分木作工作室也會出貨外縣市，只是配送費用和方式有所不同。買之前確認清楚「配送到府」還是「需自行安排搬運」即可。

**Q5：第一次買實木家具，最容易踩的雷是什麼？**

最常見的是「只看外觀不問木種」。同樣是木桌，集成材和全實木的耐用度和保養需求差很多，但外觀可能長得幾乎一樣。建議買之前一定要問清楚：**木材種類是什麼？是全實木、集成材還是貼皮板？**
