## 先想清楚：你的家具「值得賣」嗎？ {#值得賣嗎}

不是所有家具都值得花時間賣。先做個快速評估：

| 狀況 | 建議做法 |
|------|--------|
| 購入未超過 10 年、外觀良好 | 可以賣，值得花時間拍照上架 |
| 購入超過 10 年、品牌不知名 | 難賣，收購業者也可能拒絕 |
| 實木家具、知名品牌（詩肯、優渥等） | 有殘值，優先試自售 |
| 貼皮/板材家具、有明顯損傷 | 免費送人 > 花時間賣 |
| IKEA 自組系列 | 難賣，收購業者多半拒絕；試 FB 社團免費贈送 |
| 有蟲蛀、發霉嚴重 | 直接清運，不要浪費時間 |

## 估價邏輯：業者怎麼看你的家具 {#估價邏輯}

台灣二手家具收購業者（例如格上、伯樂、晶選等）在估價時，主要看四個維度：

### 1. 外觀完整度
第一眼決定一切。業者會確認：
- 表面有無無法去除的髒污
- 有無不可修復的破損（裂縫、凹陷）
- 結構是否穩固

**小技巧**：出售前做基本清潔、用濕布擦拭，有時候可以提升印象分。

### 2. 材質
實木（尤其是硬木：柚木、檜木、胡桃木、花梨木）保值性遠高於板材。業者看材質判斷能否再轉賣。

### 3. 品牌知名度
品牌家具較容易轉手。台灣知名實木品牌的二手品，收購業者接受度較高；無品牌或雜牌較難賣。

### 4. 損耗程度
使用年限、零件是否完整、功能是否正常。床架有缺零件、桌腳鬆動這類問題都會影響報價。

## 自己定價：幾折才合理？ {#自己定價}

根據台灣二手家具市場的一般行情（非官方標準，依實際狀況浮動）：

| 家具類型 | 市場行情（原價%） |
|---------|----------------|
| 實木家具（硬木，保存良好） | 約 20–40% |
| 一般品牌板材家具（5年內） | 約 10–20% |
| 一般家具（使用超過 5 年） | 約 5–15% |
| IKEA 組裝系列 | 難估，依市場供需 |

> 資料來源：格上二手家具、伯樂二手家具等業者公開說明

**定價建議**：
- 自售平台（FB、旋轉）：可定在行情上緣，預留議價空間約 10–15%
- 賣給收購業者：行情偏保守，但省時省力
- 急售（搬家倒數）：直接定行情下緣，標明「限期賣出」

## 拍照教學：讓家具看起來值得買 {#拍照教學}

拍照是二手銷售成敗的關鍵。很多好家具賣不出去，只是因為照片太差。

### 基本原則

**光線**：在自然光下拍（靠近窗戶），避免頭頂黃燈，那會讓家具看起來舊又黃。

**背景**：素色牆面或地板最好。雜亂的客廳背景是大忌——買家會看背景而不是家具。

**角度**：至少拍三個角度：
1. 正面整體（讓人看到全貌）
2. 斜 45 度（呈現立體感）
3. 細節特寫（木紋、五金、桌腳）

**瑕疵要拍**：主動拍出刮痕、污漬或使用痕跡，並在文字說明中標注。這不是讓自己被嫌棄，而是**建立信任**——誠實的賣家成交率更高，糾紛更少。

### 實木家具加分技巧

實木的紋路是最大賣點，拍照時：
- 讓側光打在桌面上，木紋會更立體
- 用濕布稍微擦拭後拍照，木紋更清晰
- 標明木種（胡桃木、橡木等），有標等於加分

## 出售管道比較：哪裡最適合你 {#出售管道}

### 台灣主要二手平台一覽

| 平台 | 特性 | 手續費 | 適合什麼 |
|------|------|--------|---------|
| **FB 社團 / Marketplace** | 在地化、免費、客群廣 | 0（若免費贈送不限） | 大型家具首選（能面交） |
| **旋轉拍賣（Carousell）** | 二手專門、客群買意明確 | 約 2% | 中小型家具、裝飾品 |
| **蝦皮拍賣** | 流量大，可宅配 | 3.5–10.5%（依方案） | 能宅配的中小件 |
| **Yahoo 拍賣** | 老平台、競標功能 | 約 2.49% | 有收藏價值的物件 |
| **PTT LivingGoods 板** | 精準受眾（裝潢族） | 0 | 品質好的實木家具 |
| **591 居家台** | 搬家族聚集 | 0 | 租屋族常用物件 |

### 建議策略

**大型實木家具**（餐桌、書桌、床架）：
1. 先發 **FB 在地社團**（「[縣市] 二手家具買賣」「搬家贈品社團」）
2. 同時掛 **旋轉拍賣**
3. 沒有成交 → 再考慮找收購業者或降價

**中小型物件**（椅子、小邊桌、置物架）：
1. **旋轉拍賣** 或 **蝦皮**（可宅配）
2. 沒反應兩週後，試 FB Marketplace

**限時急售**：
- FB 社團標明「搬家倒數 X 天，面交自取」
- 可接受較低價，但省去等待和議價時間

## 賣給收購業者 vs 自己賣：選哪個？ {#收購vs自售}

| | 自己賣（平台） | 賣給收購業者 |
|-|--------------|------------|
| 售價 | 較高（行情上緣） | 較低（業者留利潤空間） |
| 時間 | 需等人、需議價 | 到府評估後快速成交 |
| 搬運 | 自行安排或買家自取 | 業者負責搬出 |
| 適合 | 不趕時間、家具有保存價值 | 搬家倒計時、大量物件要清空 |

**實務建議**：先在 FB 社團試賣 **兩週**，若無人問，再轉向收購業者。收購業者可同時詢問多家，比價後再決定。

台灣有提供到府收購服務的業者（如格上、晶選、伯樂等），多半可以事先拍照網路估價，再確認到府時間，不需要等到現場才知道價格。

## 實木家具特別注意事項 {#實木特別注意}

實木家具出售，有幾點特別值得注意：

### 標清木種能提高成交率
「實木餐桌」和「黑胡桃木實木餐桌 4cm 板厚」是完全不同的說服力。知道木種的，請務必標明。

### 保養狀況是重要資訊
有定期上蠟、推油養護的桌子，值得在貼文說明。這是買家判斷品質的參考。

### 木頭的自然紋路不是「瑕疵」
活節、木紋變化、些微色差是實木的自然特性，不必刻意隱瞞。很多買家反而喜歡天然紋路，建議在文字中說明「此為實木自然紋路」，避免到現場後買家有誤解。

### 拆卸和搬運
大型實木餐桌往往超過 80 公斤，搬運是門檻。自售時，建議在貼文標示「需自備人力搬運」或「協助拆卸桌腳」，避免買家到現場後才發現搬不走。

## 常見問題

### Q1：IKEA 的家具二手賣得出去嗎？

比較難。IKEA 組裝系列因為原始售價低、且二手市場供給多，收購業者通常拒絕。若要賣，建議在 FB 社團免費贈送或超低價快速出清，免得耽誤搬家時程。部分品相很新的 IKEA 實木系列（如 LISABO、MÖRBYLÅNGA）還是可以試試旋轉拍賣。

### Q2：二手家具賣不掉怎麼辦？

可以考慮以下選項：
1. **免費贈送**：FB「[縣市] 免費物資贈送」社團，反應通常很快
2. **捐給公益機構**：光仁基金會、各縣市慈善協會有時提供到府搬走服務
3. **付費清運**：搬家公司附加清運費，或找廢棄物清除業者
4. **實木家具翻新再用**：如果是品質好的實木，翻新後繼續用，比賣掉更划算

### Q3：收購業者會到府估價嗎？有沒有費用？

多數台灣收購業者提供免費到府估價，但部分業者若行程太遠或物件太少可能不願出門。建議先傳照片給業者做初步報價，確認有意向再約到府時間，節省雙方時間。

### Q4：自售需要簽合約嗎？

個人二手買賣通常不簽合約，但建議：
- 金融交易留下紀錄（匯款、Line Pay 等）
- 面交時在對話記錄中確認「已驗收，無異議」
- 高價物件（一萬元以上）可考慮事先簽一份簡單書面紀錄

### Q5：實木家具有辦法知道原本的木種嗎？

可以看底部或抽屜等不顯眼處，有時有木種標籤。或用以下方法初步判斷：
- 重量：實木通常比板材重很多
- 紋路連續性：實木每個角度紋路都不一樣
- 側面剖面：沒有膠合痕跡

不確定的話，拍照發給我們，通常從木紋、顏色可以八九不離十判斷出木種。
