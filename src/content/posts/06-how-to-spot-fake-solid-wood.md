## 辨識技巧一：看木紋——連續性是關鍵

**這是最快、最直觀的方法。**

實木的木紋是從樹幹中長出來的，因此紋路是「立體的」——你從桌面看到的紋路，轉到桌面側邊，那個紋路應該是連續延伸過去的，就像一件毛衣的線沿著角落繞過去一樣。

貼皮的木紋只存在於那 0.3-0.6mm 的薄皮上。當你轉到側邊：

- **如果側邊的紋路是獨立的、跟桌面「接不起來」**——你看到的是兩片不同的木皮，每一片只有表面那一層是真木。
- **如果側邊根本沒有木紋，而是顯示出橫切面的密集細碎顆粒**——那就是密集板（MDF）或夾板，直接告訴你這是貼皮。

### 貼皮快速辨識法

看桌面的四個邊角。真實木的邊角，木紋是繞過去的（如果是實心木塊，你可以看到紋路沿著弧度轉折）。貼皮家具的邊角，有時候可以看到非常細微的「分層線」——那條線就是木皮和底材的交界。

特別要看：桌角、桌腳側邊最底部、抽屜的側面內部（通常貼皮工廠懶得貼內部）。

## 辨識技巧二：敲聲音——低沉還是空洞

這個方法需要一點練習，但用幾次之後你的耳朵會自動校準。

**方法**：用指關節敲擊桌面（像敲門那樣），聽回聲。

**實木（全實木或集成材）**：
聲音低沉、厚實、短促。像敲一塊磚頭，能量被吸收，不會有回響。

**空心板或夾板核心**：
聲音較輕、帶有共鳴感、像敲著薄薄的木板。特別是大片夾板（Plywood）桌面，中間有空氣層，敲起來會有輕微的「咚咚」共鳴。

**密集板（MDF）**：
MDF 比你想像中更重、更密，聲音其實也比較低沉。所以這個方法對「貼皮＋密集板」的組合辨識效果比較差。MDF 敲起來跟集成材有點像，但你仔細聽會發現——密集板的聲音比較「悶」，不像實木那麼有「彈性」。

### 加分技巧

同時敲桌面中心和桌面靠近邊緣的位置，比較聲音差異。實心木兩處聲音一致；空心結構在中心和邊緣的共鳴會不一樣。

## 辨識技巧三：聞味道——香氣不會說謊

這是很多人忘記用的方法，但其實非常有效。

**真實木的味道**：天然木材含有精油和樹脂，有獨特的香氣——柚木有油脂香、相思木有淡淡的甜香、白橡有草木清香、胡桃木有輕微的辛香。不同樹種香氣不同，但共同點是「天然的」，不刺鼻。

**密集板（MDF）和夾板的味道**：製作過程使用大量脲醛樹脂或三聚氰胺甲醛樹脂，會持續釋放甲醛。新傢俱如果有刺鼻的「化學味」——一般描述為裝潢味、新傢俱味——這就是甲醛。

**怎麼聞**：趴下來聞桌底、聞抽屜內部、聞桌面靠近桌腳的接縫。這些地方的空氣流通差，如果有甲醛釋放，這裡會最明顯。

### 注意

上漆（PU 漆、UV 漆）的傢俱本身也有化學溶劑味，特別是剛出廠的，這個味道通常幾週後會散去。甲醛的味道比較持久，在傢俱放進室內幾個月後仍然有刺鼻感，才是真正的警訊。

## 辨識技巧四：看切面——不開口就不知道

這是最直接的方法，但也是最容易被設計遮蓋的。

**要看哪些地方**：
1. **桌面四個邊緣**（側面）：實木應該能看到連續的木纖維截面；貼皮 MDF 則看到細碎的木屑顆粒（像壓製的鋸木屑）。
2. **桌腳底部**：桌腳最底部通常有腳墊，把腳墊拿開，看截面是什麼材質。
3. **抽屜底板內側**：抽屜底板通常是夾板，這可以接受，但如果側板也是夾板或 MDF，就要注意。
4. **門板的鉸鍊孔附近**：鑽孔時往往會露出內部材質，是蜂巢紙板、MDF 還是實木一目了然。

### 高級做法：問店家

如果真的是全實木，品牌應該歡迎你看截面，甚至可以拿一小段裁切廢料給你看。如果對方支吾其詞、說「這樣會損壞商品」——這本身就是一個答案。

## 辨識技巧五：測重量——密度會說話

把傢俱抬起來，感受一下重量。

**一般參考值（以常見木種為例）**：

| 木種 | 密度（kg/m³） |
|------|-------------|
| 柚木 | 650-700 |
| 胡桃木 | 600-650 |
| 白橡木 | 700-750 |
| 相思木 | 750-850 |
| 松木 | 450-550 |
| 密集板（MDF） | 650-800 |
| 夾板（Plywood） | 500-700 |

你可以看到，問題來了：**密集板的密度和實木非常接近，所以「很重就是實木」這個說法是錯的。**

但有一種情況重量很有用：如果一件傢俱標榜是「實木白橡木」但輕得像紙板，那肯定有問題——因為白橡木密度很高，不可能這麼輕（很可能是空心結構或夾板）。

所以重量是「排除法」——**太輕一定有問題，但夠重不代表一定是實木**。

## 常見業務話術拆解

### 話術一：「這是北美原木，進口的喔！」

**拆解**：「北美進口」確實存在——白橡木、胡桃木多數從北美進口。但「北美進口」只說明了木種可能的來源，不代表這件傢俱的工法和用料是全實木。可能是北美進口木皮、貼在台灣製的 MDF 上。追問：「整件傢俱的木料是實心的嗎？切面可以看嗎？」

### 話術二：「環保板材，比實木更好，不含甲醛！」

**拆解**：「環保板材」通常是指使用 F☆☆☆☆（日本標準）或 E0 等級的低甲醛板材，確實比一般 MDF 甲醛含量低，但它依然不是實木。這種說法的目的是把「貼皮板材」包裝成比實木更好的選擇——有其合理的地方，但要你知道你買的是什麼。

### 話術三：「這個接縫是故意做的，日式工法！」

**拆解**：如果你指出接縫處有明顯分層，業務說這是「工藝特色」，這時候要仔細看。真正的日式拼接（指接榫）確實有接縫，但接縫兩側的材質是相同的——都是木頭。如果接縫一邊是木頭、另一邊露出密集板截面，這就是貼皮包邊，不是工藝特色。

### 話術四：「這是實木的，只是外面有上漆，所以看起來不一樣。」

**拆解**：漆可以遮蓋很多東西。高亮 PU 漆特別容易掩蓋木紋的真偽——因為強光反射讓人難以辨認木紋的深度。遇到這種情況，回到「敲聲音」和「看切面」兩個方法，不依賴眼看。

### 話術五：「老師傅做的，手工的，一定比機器好！」

**拆解**：這話對，但跟材質無關。老師傅可以用密集板做出工藝精良的貼皮傢俱，機器也可以加工出品質優秀的實木。「手工」是工藝的描述，不是材質的保證。

## 完整驗貨 Checklist

下次去傢俱店，把這個 Checklist 截圖帶著用：

### 📋 現場辨識清單

**外觀檢查**
- [ ] 桌面與桌面側邊的木紋是否連續延伸（轉角不斷）？
- [ ] 桌腳側邊是否有連續木紋（非貼皮邊條）？
- [ ] 抽屜側板內部是否為木纖維截面（非 MDF 顆粒）？
- [ ] 桌面四周邊緣有無「拼接線」痕跡（木皮收邊）？

**聲音測試**
- [ ] 指關節敲桌面，聲音是否低沉厚實？
- [ ] 桌面中心與邊緣的敲擊聲音是否一致？

**氣味測試**
- [ ] 趴近桌底、抽屜內部，是否有刺鼻化學味？
- [ ] 有沒有自然木材的香氣？

**重量感受**
- [ ] 單人能否輕鬆抬起一側？（同尺寸實木桌通常需要兩人）

**詢問確認**
- [ ] 問：這件傢俱的材質是全實木還是集成材？
- [ ] 問：可以讓我看一下側面截面嗎？
- [ ] 問：木料有做窯乾（Kiln-dried）處理嗎？
- [ ] 問：保固是幾年？有書面條款嗎？

**高分加測**
- [ ] 用指甲輕刮側邊不顯眼處——木頭有纖維感，PVC 貼膜有白邊
- [ ] 問有無木材含水率測試報告

## 常見問題

**Q1：集成材算假實木嗎？**

A：不算。集成材是用真實木料拼接而成，每一塊材料都是真實的木頭，只是以榫接或膠合方式組成更大的板材。它跟貼皮（表面是薄皮、核心是密集板）本質上不同。市面上很多「假實木」爭議，其實是把集成材和全實木的差異誤解成「真假」問題。

**Q2：朋友說可以用磁鐵吸來分辨，是真的嗎？**

A：有一定道理，但要知道其局限。部分密集板和夾板的製程中會摻入鐵系材料，但這不是普遍規律。磁鐵能吸到的話確定有金屬成分（但金屬螺絲、五金件本來就有），吸不到不代表就是實木。這個方法太不穩定，不推薦作為主要辨識手段。

**Q3：買傢俱前要求看截面，會不會很奇怪？**

A：不奇怪。一個誠信的傢俱業者歡迎你問，因為他們知道自己的料是好的。如果對方對這個問題表現得很防衛，這個反應本身就告訴了你答案。你花幾萬甚至十幾萬買一件傢俱，有權利知道你在買什麼。

**Q4：如果我已經買了，發現是假實木，能退貨嗎？**

A：依據消費者保護法，如果業者標示不實（明說全實木、賣的卻是貼皮），這屬於詐欺消費，可以申請退貨退款。建議保留購買憑證、業務口頭或書面承諾的截圖，向消費者保護委員會申訴。台灣的消費者保護法規在這方面站你這邊。

**Q5：有沒有辦法不去現場，從照片判斷？**

A：有限制，但有些訊號可以觀察。高解析度照片中，你可以看桌面側邊、抽屜內部、桌腳底部的截面——如果業者特意迴避這些角度的照片，通常有原因。另外，「渲染圖」比真實照片更難判斷。記住：線上購買實木家具，選擇有實體展間的品牌才是保障。
