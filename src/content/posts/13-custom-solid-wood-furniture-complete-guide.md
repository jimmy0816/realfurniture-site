## 為什麼要客製化？ {#為什麼要客製化}

成品家具有標準尺寸、固定設計、快速交貨的優勢，但客製化的價值在於：

| 需求 | 成品家具 | 客製化家具 |
|------|---------|----------|
| 尺寸剛好卡住 | ✗ 只能妥協 | ✓ 完全吻合空間 |
| 想要特定木種 | ✗ 選擇有限 | ✓ 可指定材料 |
| 需要特殊功能 | ✗ 規格固定 | ✓ 依需求設計 |
| 風格高度一致 | △ 看運氣 | ✓ 整體搭配 |
| 預算 | 便宜~中等 | 中等~高 |
| 交期 | 1-2 週 | 1-2 個月 |

**適合客製化的情境：**
- 空間尺寸特殊（斜角、樑柱、窗邊）
- 想要完整的風格一致性（如整套書房家具同木種）
- 特定功能需求（如內嵌插座、隱藏式抽屜）
- 對材料、工法有明確偏好

**不適合客製化的情境：**
- 預算有限且時間緊迫
- 對設計沒有特定想法，「看起來 OK 就好」
- 租屋族，未來可能搬家

## 客製化流程完整拆解（7 步驟） {#流程拆解}

### 步驟 1：初步需求溝通

**你要做的事：**
- 決定要訂製的家具類型（餐桌、書桌、櫃體等）
- 準備尺寸資料（自行丈量或請設計師協助）
- 蒐集參考圖片或簡易手稿
- 確認木種偏好（胡桃木、橡木、柚木等）

**與工坊聯絡時要說的：**
- 「我想訂做一張 XXX，尺寸大約 XXX 公分」
- 「希望木種是 XXX，表面處理偏好推油 / 上漆」
- 「附上參考圖片，想要類似這種設計」
- 「預算範圍大約在 X 萬以內」

**工坊會回應的：**
- 製作可行性（有些設計因結構或材料限制無法實現）
- 初步價格範圍
- 大致交期

💡 **里爾提醒：** 這階段不用太精確，重點是讓工坊知道你的方向，才能判斷是否接案。

---

### 步驟 2：現場丈量與環境評估

**為什麼需要現場丈量？**

即使你已經量過尺寸，專業工坊通常會要求到現場確認：
- **天花板高度**（影響櫃體設計）
- **電梯 / 樓梯尺寸**（大型桌板能不能運上去？）
- **走道寬度**（家具能否順利進門？）
- **牆面材質**（如果要做壁掛櫃，需評估承重）
- **光線 / 空間氛圍**（實際看過才能確認顏色與風格）

**案例：**
有位客戶訂了一張 220 公分的大餐桌，結果交貨時發現電梯放不下，最後只能拆成兩塊板材現場組裝，多花了工資和時間。提前評估可以避免這種狀況。

---

### 步驟 3：草圖與正式報價

**工坊會提供：**
- 手繪或 3D 草圖（顯示大致外觀）
- 詳細報價單，包含：
  - 木材種類與材積（才數）
  - 五金配件
  - 表面處理方式（推油 / 上漆）
  - 運費與安裝費
  - 預估交期

**報價單必須有的項目：**

| 項目 | 說明 |
|------|------|
| 木材 | 木種 + 才數 + 單價 |
| 工法 | 榫卯 / 五金接合 |
| 表面處理 | 推油 / 上漆（品牌與等級）|
| 五金 | 滑軌、鉸鍊、把手等 |
| 運送 | 運費與樓層費 |
| 安裝 | 現場組裝與調整 |
| 保固 | 保固期限與範圍 |

⚠️ **紅線：** 如果報價單只有一個總價數字，沒有拆分項目，建議要求補充明細。模糊報價是後續糾紛的主要來源。

---

### 步驟 4：確認設計圖與簽約

工坊會根據草圖討論結果，提供正式設計圖。

**你要確認的：**
- ✅ 尺寸標註是否正確
- ✅ 木材紋路方向（如桌面橫紋 vs 直紋）
- ✅ 抽屜、櫃門的開啟方式
- ✅ 五金位置與顏色
- ✅ 表面處理方式

**簽約時要注意的：**
- 總價與付款方式（通常是訂金 30-50%，尾款交貨時付清）
- 交期承諾（並註明延遲的處理方式）
- 保固條款（結構、五金、表面處理各保固多久）
- 驗收標準（允許的色差範圍、木材天然特徵如節點是否接受）

💡 **里爾提醒：** 實木有天然色差與紋路變化，這是正常現象。如果你對色差極度敏感，簽約前務必溝通清楚，並在合約中註明「若色差過大可退換」的條款。

---

### 步驟 5：製作期（30-60 天）

**典型製作流程：**
1. 木材備料與窯乾檢驗
2. 裁切、刨平、定厚
3. 榫卯加工或五金孔位加工
4. 組裝與結構測試
5. 打磨（粗磨 → 細磨 → 拋光）
6. 表面處理（推油或上漆，多層施作）
7. 最終檢驗

**交期多久才合理？**

| 家具類型 | 參考交期 |
|---------|---------|
| 簡單餐桌（無抽屜） | 30-45 天 |
| 書桌（含抽屜） | 35-50 天 |
| 整組櫃體 | 45-60 天 |
| 複雜設計（如多層櫃、特殊造型）| 60 天以上 |

⚠️ **注意：** 若遇到農曆年前、梅雨季、或大型案件排隊，交期可能拉長。下訂前務必確認。

---

### 步驟 6：交貨與搬運

**交貨前你要做的：**
- 確認家中動線淨空
- 通知大樓管理處（預約電梯、登記搬運時間）
- 確認安裝位置附近沒有障礙物

**搬運時常見問題：**
- 大型板材無法進電梯 → 需拆分或走樓梯（可能加收費用）
- 樓層高需使用吊車 → 需事先評估與報價
- 地板保護 → 確認工坊有鋪保護墊

---

### 步驟 7：驗收

**驗收必檢的 5 個項目：**

#### 1. 外觀檢查
- 木紋方向與顏色是否符合設計圖
- 表面是否平整、無明顯刮痕
- 邊角是否光滑、無毛刺

#### 2. 結構檢查
- 桌/櫃是否穩固，不晃動
- 抽屜開關是否順暢
- 櫃門鉸鍊是否對齊

#### 3. 五金檢查
- 滑軌、把手是否牢固
- 開關次數測試（開關 10 次確認順暢）

#### 4. 尺寸核對
- 實際尺寸與圖面是否一致
- 與空間的配合度（如桌腳是否剛好卡樑柱）

#### 5. 表面處理
- 推油家具：觸感是否細膩、無油漬
- 上漆家具：漆面是否均勻、無氣泡

💡 **里爾提醒：** 驗收當場發現問題要立刻反映，並拍照記錄。簽收後若發現非結構性小瑕疵，部分工坊可能不再免費處理。

## 報價怎麼算？才數、材積、工法的影響 {#報價方式}

### 木材計價單位：才

台灣木材買賣常用「才」為單位：

**1 才 = 1 台尺 × 1 台尺 × 1 台寸 = 0.0028 立方公尺**

例如：
- 一塊 180 公分 × 90 公分 × 4 公分的桌板
- 換算：約 6 台尺 × 3 台尺 × 1.2 台寸 = **21.6 才**
- 若胡桃木單價 1,500 元/才，這塊板材成本約 **32,400 元**

### 影響報價的因素

| 因素 | 影響 |
|------|------|
| 木種 | 胡桃木 > 橡木 > 松木 |
| 材積 | 越大越貴 |
| 工法 | 榫卯 > 五金接合 |
| 表面處理 | 多層上漆 > 推油 |
| 設計複雜度 | 複雜造型、多抽屜加價 |
| 五金等級 | 進口滑軌 > 國產滑軌 |

**參考價格範圍（僅供參考，依設計與材料而異）：**
- 簡單實木餐桌（150×80 cm）：3-8 萬
- 實木書桌（含抽屜）：4-10 萬
- 訂製衣櫃（木作）：1,500-2,500 元/尺

⚠️ **紅線：** 以上價格僅為市場概略範圍，實際報價會依木種、工法、設計細節而有大幅差異。

## 溝通時必須提供的資訊清單 {#溝通清單}

為了讓工坊準確報價與設計，建議準備以下資訊：

### 基本資訊
- [ ] 家具類型（餐桌 / 書桌 / 櫃體 / 床架等）
- [ ] 尺寸需求（長 × 寬 × 高）
- [ ] 使用情境（幾人使用、放什麼物品）

### 材料偏好
- [ ] 木種（胡桃木 / 橡木 / 柚木 / 相思木等）
- [ ] 表面處理（推油 / 上漆）
- [ ] 五金等級（一般 / 進口品牌）

### 設計風格
- [ ] 參考圖片（Pinterest、Instagram 或成品照）
- [ ] 顏色偏好（深色 / 淺色 / 自然原色）
- [ ] 特殊功能需求（內嵌插座 / 隱藏式抽屜 / 可延伸桌板等）

### 環境限制
- [ ] 空間尺寸與格局圖
- [ ] 電梯 / 樓梯尺寸
- [ ] 天花板高度（若做櫃體）
- [ ] 牆面材質（若要壁掛）

### 預算與時間
- [ ] 預算範圍
- [ ] 希望交貨時間

## 交期多久才合理？ {#交期}

**影響交期的因素：**

1. **設計複雜度**：簡單桌面比多抽屜櫃快
2. **木材備料**：若指定稀有木種，可能需等待進貨
3. **工坊排程**：年前、旺季排隊時間長
4. **表面處理**：上漆需多層施作，每層需乾燥時間
5. **天氣**：梅雨季影響上漆與乾燥

**合理交期參考（以一般實木家具為例）：**
- 簡單餐桌：30-45 天
- 含抽屜書桌：35-50 天
- 櫃體：45-60 天
- 大型或複雜設計：60 天以上

⚠️ **注意：** 如果工坊承諾「2 週交貨」，除非是極簡設計，否則要留意品質是否被壓縮。

## 驗收時必檢的 5 個項目 {#驗收}

前面已提過，這裡再整理成 Checklist：

### ✅ 驗收清單

**外觀：**
- [ ] 木紋與顏色符合預期
- [ ] 表面平整無刮痕
- [ ] 邊角光滑無毛刺
- [ ] 無明顯色差（在可接受範圍內）

**結構：**
- [ ] 桌面 / 櫃體不晃動
- [ ] 抽屜開關順暢
- [ ] 櫃門對齊、無歪斜

**五金：**
- [ ] 滑軌、鉸鍊牢固
- [ ] 把手位置正確
- [ ] 開關測試順暢

**尺寸：**
- [ ] 與設計圖一致

**表面處理：**
- [ ] 推油：觸感細膩、無油漬
- [ ] 上漆：漆面均勻、無氣泡或流痕

💡 **發現問題怎麼辦？**
- 當場反映並拍照
- 要求工坊載回調整或現場修復
- 確認修復時間與方式
- 修復完成後再簽收

## 小結：客製化的核心是「溝通」

客製化實木家具不是一場交易，而是一次合作。

工坊需要你清楚表達需求，你需要工坊提供專業建議。越早確認細節、越完整的溝通，越能避免後續糾紛。

如果你不確定某個設計是否可行、預算是否合理，建議多問幾家工坊，比較報價與設計方案。不同工坊的專長、材料來源、工法偏好都不同，找到理念相近的合作對象，才能做出真正滿意的家具。

## 常見問題

**Q1：訂金付了之後可以退嗎？**  
A：這取決於合約條款。一般來說，若設計圖尚未確認、尚未進入製作階段，部分工坊可能願意扣除設計費後退還部分訂金。但若已下料製作，因客製化家具無法轉售，通常不接受退訂。建議簽約前務必確認退訂條款，並在設計圖確認階段反覆檢視，避免事後後悔。

**Q2：客製化家具比成品貴多少？**  
A：這沒有標準答案，因為客製化的價格主要由木種、尺寸、工法、設計複雜度決定。一般而言，若選擇相同木種與類似設計，客製化可能比成品貴 20-50%，因為包含了設計費、手工費、以及無法量產的成本。但如果你的需求特殊（如尺寸剛好卡在成品規格之間），客製化反而可能更符合預算，因為不需要妥協購買過大或過小的家具。

**Q3：可以只提供照片請工坊照做嗎？**  
A：可以，但需要注意幾點：(1) 提供的照片要有多角度、清楚的細節，(2) 說明你想保留的設計元素與可調整的部分，(3) 確認原設計的尺寸與結構是否適合你的空間。有些設計看起來美觀，但可能在實際使用上不夠實用，專業工坊通常會提供調整建議，不要只是一味照抄。

**Q4：交貨延遲怎麼辦？**  
A：首先確認合約中是否有註明交期與延遲處理方式。一般來說，若延遲是因不可抗力（如颱風、材料短缺），工坊通常會提前通知並協調新交期。若延遲責任在工坊（如排程失誤），可以要求賠償或折扣。建議在簽約時就確認「若延遲超過 X 天，每日折抵 X 元」的條款，保障雙方權益。

**Q5：實木家具客製化後還需要保養嗎？**  
A：需要。客製化家具與成品家具的保養方式一樣，取決於表面處理方式。推油家具需要每半年至一年補油，上漆家具則較省事，只需定期擦拭。無論哪種處理，都要避免長期直射陽光、冷氣直吹、以及過度潮濕的環境。建議交貨時向工坊索取保養說明書，並詢問是否提供定期保養服務。
