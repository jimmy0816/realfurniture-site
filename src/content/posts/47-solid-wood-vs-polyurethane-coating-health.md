## 「波麗塗裝」是什麼意思？

在台灣家具業，「波麗」通常指 **PU 漆（Polyurethane，聚氨酯漆）**，也叫做「優麗旦」。這是一種以有機溶劑為基底的塗料，塗裝完成後能在木材表面形成一層厚實、亮滑的漆膜，光澤感強，耐磨耐刮。

**重要觀念**：波麗塗裝可以塗在實木上，也可以塗在密集板或夾板上。所以「波麗塗裝家具」不等於「非實木家具」——問題在塗料本身，不在木材基材。

消費者容易誤以為「實木 vs 波麗」是兩種家具類別的對比，其實更精確的說法是：

> **天然工法塗裝的實木家具** vs **油性 PU 漆重塗裝家具**

這才是健康層面真正需要比較的問題。

## 塗裝的健康問題：VOC 與甲醛

### VOC 是什麼？

VOC（Volatile Organic Compounds，揮發性有機化合物）是漆料施工後會蒸發到空氣中的有機物質總稱。範圍很廣，包括甲苯、苯乙烯、乙苯、丙烯酸等數十種化學物。

根據台灣環保署規範，室內 **TVOC（揮發性有機物總量）標準值 ≤ 0.56 mg/m³**。

傳統油性 PU 波麗漆的溶劑揮發期約 **7 到 30 天**——聞起來有濃烈的油漆味。但這只是「你感覺得到的部分」，某些無味的化合物可能持續更久。

### 甲醛哪裡來？

關於甲醛，有一個關鍵澄清：

**塗料本身不是甲醛的主要來源**，真正的問題在於——

| 甲醛來源 | 說明 |
|---------|------|
| 板材膠合層 | 夾板、密集板、粒片板的黏著劑 |
| 系統家具板材 | 等級差的 E2 板材可釋放甲醛長達數年 |
| 家具內部填充物 | 床墊、沙發海棉 |
| **PU 波麗漆本身** | VOC 含量高，但甲醛含量相對低 |

根據台灣 CNS 標準及室內空氣品質管理法，**室內甲醛濃度標準 ≤ 0.08 ppm**（與歐美標準相同）。世界衛生組織（WHO）旗下的國際癌症研究機構（IARC）已將甲醛列為**第 1 類致癌物**，長期暴露與鼻咽癌及白血病有關。

所以，選家具時要同時注意：**板材等級**（甲醛問題）＋**塗裝工法**（VOC 問題）。

## 四種主要塗裝方式完整比較

市面上常見的木器塗裝工法，由健康友善程度由低到高排列：

### 1. 傳統油性 PU 波麗漆

最常見的工廠量產塗裝，也就是俗稱的「波麗面」。

| 項目 | 說明 |
|-----|------|
| 成分 | 聚氨酯樹脂 + 有機溶劑 |
| VOC 含量 | **高**（施工期間氣味明顯）|
| 外觀 | 高光澤、漆膜厚、色澤均一 |
| 耐用性 | 耐磨、耐刮，保護力強 |
| 環保性 | ❌ 最差，歐盟已限制室內使用 |
| 適用情境 | 低至中價位量產家具、工廠出貨速度快 |

**健康疑慮**：施工過程中工人暴露在高濃度 VOC 環境；成品出廠初期，殘留溶劑仍持續揮發。對氣道敏感者、孕婦、嬰幼兒風險較高。

值得一提：歐盟《裝飾塗料指導方針》已明確鼓勵降低室內塗裝 VOC 含量，並逐步限制油性漆的室內應用。台灣市場雖然落後歐盟，但健康意識也在逐漸抬頭。

---

### 2. NC 硝化纖維漆（硝基漆）

比 PU 漆更早期的塗裝工法，部分傳統工廠仍在使用。

| 項目 | 說明 |
|-----|------|
| 成分 | 硝化纖維素 + 有機溶劑 |
| VOC 含量 | **高**，揮發快速 |
| 外觀 | 漆膜薄，透明度高 |
| 耐用性 | 耐磨性較差，容易黃化變色 |
| 環保性 | ❌ 差，且對溶劑和化學品抵抗性弱 |
| 適用情境 | 中低價位裝飾性家具 |

相比 PU，NC 漆揮發更快，短期刺激較強，長期殘留反而較少。但整體環保等級仍不理想。

---

### 3. 水性漆（Water-based Coating）

近年台灣高端實木家具品牌愈來愈多轉用水性漆，是目前**性能與環保的最佳平衡點**。

| 項目 | 說明 |
|-----|------|
| 成分 | 水性樹脂（以水為溶劑） |
| VOC 含量 | **極低**，幾乎無刺鼻味 |
| 外觀 | 保護力佳，色澤自然 |
| 耐用性 | 稍遜 PU，但現代配方已大幅提升 |
| 環保性 | ✅ 佳，符合歐盟標準 |
| 適用情境 | 兒童房、孕婦居家、追求健康的買家 |

水性漆乾燥速度快、施工過程對工人傷害小、成品揮發物極低。缺點是塗裝成本比油性 PU 高，工法更繁瑣，所以仍有部分廠商以成本考量選擇不使用。

---

### 4. 木蠟油 / 護木油（Natural Oil & Wax）

最接近「零化學」的塗裝選擇，由天然植物油脂製成，代表性品牌如德國 Osmo、AURO 等。

| 項目 | 說明 |
|-----|------|
| 成分 | 天然植物油、天然蠟 |
| VOC 含量 | **最低**，部分產品幾近於零 |
| 外觀 | 保留木材天然紋理，觸感溫潤 |
| 耐用性 | 中等，需定期（1-2 年）補塗保養 |
| 環保性 | ✅✅ 最佳，對人體及環境最友善 |
| 適用情境 | 追求天然質感、有幼兒或過敏體質的家庭 |

木蠟油滲透木材內部而非成膜覆蓋，讓木材仍能「呼吸」，保留溫濕度調節的天然特性。缺點是耐磨性不如 PU 漆，需要定期維護。

## 台灣的標準規範怎麼說？

台灣對木材板材的甲醛釋放量，採 CNS 國家標準，從低到高分為：

| 等級 | 甲醛釋放量 | 備注 |
|-----|----------|------|
| F1（最低）| ≤ 0.5 mg/L | 相當於日本 F★★★★（四星） |
| F2 | ≤ 1.5 mg/L | 一般系統家具常見 |
| F3 | ≤ 5.0 mg/L | 不建議用於室內 |

對應室內空氣：
- **甲醛濃度標準 ≤ 0.08 ppm**（台灣室內空氣品質管理法）
- **TVOC 標準 ≤ 0.56 mg/m³**（台北市環保局規範）

選購建議：板材至少達到 **F1 等級或日本 F☆☆☆☆**，塗裝選擇水性漆以上，才能確保長期居住健康。

## 怎麼選才對？買前必問清單

不管你在哪個城市採購、線上還是實體，跟家具品牌確認這幾件事：

**問塗裝：**
1. 這件家具用什麼塗裝工法？是 PU 油性漆、水性漆，還是木蠟油？
2. 有沒有 SGS 或相關塗裝 VOC 檢測報告？

**問板材（如果有用到板材）：**
3. 板材甲醛等級是 F1 還是 F2？有沒有 CNS 或 F☆☆☆☆ 認證？
4. 板材膠合劑是否使用 MDI 膠或低甲醛配方？

**問使用狀況：**
5. 新家具建議開窗通風多久？

品牌願意清楚回答這些問題，通常就是有料、有自信的選擇。

## 實際決策怎麼做？

根據你的需求對號入座：

**有嬰幼兒 / 孕婦 / 過敏體質** → 優先選**木蠟油**或**水性漆**塗裝，板材要求 F1 等級以上。

**預算有限、家具不放臥室** → PU 波麗漆家具可接受，但新家具務必通風 2-4 週再頻繁使用。

**追求質感 + 長期投資** → **全實木 + 木蠟油**，初期保養成本高，但健康、耐用、可修復。

**租屋族 / 不打算長住** → 水性漆的量產實木家具，平衡健康與預算。

## 一個常被忽略的觀念

**家具本身的木材不是問題，問題在膠和漆。**

全實木家具如果使用 PU 波麗重塗裝，健康風險並不比板材家具低多少。反過來，使用 F1 等級板材加上水性漆的系統家具，可能比「實木但波麗塗裝」更友善。

結論：與其只盯著「實木 vs 非實木」，不如多問一句：**「這件家具用什麼塗？有沒有第三方檢測？」**

## 常見問題

**Q1：買回來的波麗塗裝家具，要等多久才能放心使用？**

新家具開封後，建議放置在通風良好的空間，開窗保持空氣流通。油性 PU 漆的 VOC 揮發期約 **7 至 30 天**，完全穩定約需一個月。若有嬰幼兒或孕婦同住，建議延長至 4-8 週。最安全的做法是要求廠商提供 SGS 檢測報告，確認 VOC 及甲醛含量符合台灣室內標準。

**Q2：木蠟油家具需要多久保養一次？台灣氣候適合嗎？**

一般建議每 **1 至 2 年**補塗一次木蠟油，高使用頻率的桌面可縮短至 1 年一次。台灣氣候較潮濕，木蠟油塗裝能讓木材維持呼吸調節的能力，反而比封閉式漆膜更適合台灣環境。日常用微濕布擦拭即可，避免強效清潔劑。

**Q3：水性漆和木蠟油，哪個更耐用？**

論純粹的**耐磨耐刮**，水性漆優於木蠟油；論**長期修復性**，木蠟油勝出。木蠟油家具出現刮痕或污漬，只需局部補塗即可修復，不需整張桌面重新處理。水性漆若漆膜破損，修補較繁瑣。各有優劣，取決於你的使用習慣和願意投入的保養時間。

**Q4：家具商說「環保塗裝」，這有規範標準嗎？**

「環保塗裝」是行銷用語，沒有法定定義。你要追問的是**具體認證**，例如：台灣 CNS F1 等級、日本 F☆☆☆☆（四星）、德國 Blue Angel 標章，或 SGS 第三方 VOC 檢測報告。有具體文件背書的，才算數。

**Q5：小孩房的家具特別需要注意哪一點？**

兒童房的建議優先順序：① 板材用 **F1 等級以上**（甲醛問題）② 塗裝選**水性漆或木蠟油**（VOC 問題）③ **避免雙面貼皮的封閉式系統家具**（膠合層面積大，甲醛累積多）。若預算有限，至少做到板材等級達標，並確保通風充足。
