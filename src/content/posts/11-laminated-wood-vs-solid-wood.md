## 集成材是什麼？ {#集成材是什麼}

集成材（英文：Laminated Wood，又稱實木拼板、實木指接板、實木集成板）是把較小塊的實木木料，經過挑選、裁切、指接（Finger Joint）或平接（Butt Joint），再用無毒或低毒膠合劑黏合壓製成的木板。

**關鍵概念：集成材依然是「真正的木頭」，只是不是一整塊。**

市場上有些人會說集成材是「假實木」，這個說法並不準確。集成材用的是真實的木料，跟密集板（木屑壓製）或夾板（薄木片交疊）的結構完全不同。正確的分類應該是：

| 材料 | 原料 | 是否為實木 |
|------|------|-----------|
| 原木（全實木） | 整塊原木裁切 | ✅ 是 |
| 集成材 | 小塊實木拼接膠合 | ✅ 是（拼接實木） |
| 木心板 | 實木條+薄夾板 | ⚠️ 部分 |
| 夾板（合板） | 薄木片交疊膠合 | ⚠️ 部分 |
| 塑合板 | 木屑+膠合劑 | ❌ 否 |
| 密集板（MDF） | 木粉+樹脂 | ❌ 否 |

## 集成材的製作流程 {#製作流程}

一塊品質好的集成材，製程大致如下：

**1. 原木裁切**
將大根原木鋸成小截面木料，通常是板狀或條狀。這個階段會篩除有明顯缺陷的部分（大裂紋、蟲蛀），保留品質穩定的木料。

**2. 窯乾（Kiln-drying）**
裁切後的小木料送入窯中，控制溫度和濕度降低含水率。這個步驟攸關集成材的穩定性——含水率沒有控制好，拼接後仍然會變形。

**3. 指接（Finger Joint）或平接（Butt Joint）**
- **指接**：木料端部切出鋸齒狀，彼此咬合黏接，斷面像手指交扣——這是最常見的集成方式，可以看到明顯的「指接線」。
- **平接**：木料側面直接對貼，看起來較像整塊木頭，但接縫面積較小。

**4. 膠合加壓**
使用膠合劑將拼好的木料加壓固定。這裡使用的膠合劑類型，直接影響集成材的甲醛釋放量。

**5. 砂磨整平**
壓製完成後刨平、砂磨，就是最終的集成板材。

## 集成材 vs 原木：完整比較表 {#比較表}

| 比較項目 | 集成材 | 原木（全實木） |
|---------|--------|---------------|
| **外觀** | 有指接線或拼接紋路 | 天然一體紋路，木節連續 |
| **穩定性** | ✅ 較佳（多方向抵消應力） | ⚠️ 較易翹曲開裂 |
| **耐用度** | ★★★★ | ★★★★★ |
| **甲醛疑慮** | ⚠️ 有（依膠合劑等級而異） | ✅ 幾乎無 |
| **環保性** | ✅ 較佳（充分利用木料） | ⚠️ 需要較大原木 |
| **取得難易** | 容易（可大量生產） | 較困難（大面積整塊原木稀少） |
| **價格** | 較低（依木種約為原木的 50–70%） | 較高 |
| **適合風格** | 現代、北歐、工業風 | 傳家型、日式職人風 |

**穩定性為什麼集成材反而佔優？**  
原木是整塊木頭，木紋應力方向一致，溫濕度變化時容易朝同一方向翹曲。集成材因為多塊木料方向交替，應力互相抵消，整體收縮膨脹的幅度更小——這也是為什麼許多專業木工師傅做餐桌桌板時，反而會選用集成材做基底。

## 集成材的甲醛問題：F1、F2、F3 怎麼選 {#甲醛問題}

集成材最常被消費者擔心的問題，是製程中使用的**膠合劑**。不良的膠合劑會在使用過程中持續釋放甲醛，影響室內空氣品質。

台灣的木板材甲醛含量以 **CNS 標準**分級：

| 等級 | 甲醛釋放量（平均值） | 建議用途 |
|------|-------------------|---------| 
| **F1**（最低） | ≤ 0.3 mg/L | 兒童房、臥室、密閉空間 |
| **F2** | ≤ 0.5 mg/L | 一般居家家具 |
| **F3** | ≤ 1.5 mg/L | 通風良好的空間，或戶外用途 |

> 📌 **歐洲對照**：歐規 E1 等級（≤ 0.1 ppm）約對應台灣 F3 等級；日本 F☆☆☆☆（最嚴格）約對應台灣 F1。消費者在查看進口集成材時，也可能看到歐規 E1 標示。

**選購建議：**
- 兒童房、臥室家具 → 指定 **F1** 等級（或日本 F☆☆☆☆）
- 客廳餐廳等通風處 → **F2** 以上即可
- 市面上部分廠商標榜「E0」，但 E0 並非台灣官方標準，屬民間自訂，需要求廠商提供第三方檢驗報告

**實際操作：** 購買前直接詢問店家「這塊集成材是哪個甲醛等級？有沒有 SGS 或 CNS 驗證報告？」，有實力的賣家都能提供，提不出報告的就要謹慎。

## 哪些家具適合用集成材？ {#適合用途}

集成材並非「次等選擇」，它有自己最適合的應用場景：

### ✅ 適合集成材的情境

- **大面積桌板**（餐桌、書桌、工作桌）：需要穩定不翹曲，集成材穩定性佔優
- **書架、收納層板**：需要承重且不易變形
- **兒童家具**：只要選 F1 等級，安全無虞，且萬一碰撞造成傷損也容易修復
- **預算有限但追求實木質感**：集成材提供接近原木的視覺效果，價格合理
- **現代、北歐、工業風格**：指接線有時反而成為設計特色

### ⚠️ 考慮原木更合適的情境

- **傳家型家具、高端工藝訴求**：整塊木料的紋路連貫性和稀缺感，集成材無法複製
- **台灣檜木、柚木等高價值木種**：這類木種本身就有天然防腐、抗菌特性，原木最能保留這些特質
- **異形雕刻、榫卯工法主結構**：複雜的手工榫卯接頭，通常需要較大面積的整塊料

## 如何辨認集成材？現場 3 步驟 {#如何辨認}

到店裡看家具，怎麼判斷是集成材還是原木？

**步驟 1：看側邊或底部**  
集成材的側面可以看到木料拼接線；原木的側面是連續木紋。如果桌面邊緣剛好有貼皮或封邊條遮蓋，就要找底部來看。

**步驟 2：找指接線（Finger Joint 鋸齒線）**  
翻到桌面底面，集成材通常可以看到等間距的鋸齒狀接縫——這是指接工藝的特徵，不是瑕疵，而是正常製程。

**步驟 3：直接問木種 + 等級**  
「這塊集成材的木種是什麼？甲醛等級是幾級？」這兩個問題問完，產品品質差不多就清楚了。

| 你的情境 | 建議選擇 |
|---------|---------|
| 預算有限，想要實木質感 | 集成材（F2 以上） |
| 家裡有小孩，重視空氣品質 | 集成材（**F1 等級**指定） |
| 想要大面積桌板（120cm 以上）且穩定不翹 | 集成材 |
| 追求傳家型、整塊木紋、高端工藝 | 原木 |
| 喜歡柚木、檜木等珍貴木種的天然特性 | 原木 |
| 擔心未來修復、補漆 | 兩者皆可（實木修復彈性較大） |

## 常見問題 {#常見問題}

**Q1：集成材跟「實木」是一樣的東西嗎？**  
A：集成材是「實木拼板」，原料是真正的木材，不是木屑或纖維板。它屬於廣義的「實木」範疇，但跟一整塊原木（全實木）在結構上不同，外觀上可以看到指接線。

**Q2：集成材會不會比原木更容易壞？**  
A：不一定。集成材因為多向排列木料，穩定性通常優於同規格的原木板，不容易翹曲或開裂。耐用度的關鍵更多在於「木種硬度」和「表面處理方式」，而非是否為集成材。

**Q3：集成材有甲醛風險嗎？**  
A：有，但可管控。關鍵是膠合劑等級。選購時指定 F1 或 F2（台灣 CNS 標準）、或歐規 E1 以上，並要求廠商提供第三方驗證報告，可大幅降低甲醛暴露風險。

**Q4：集成材可以訂製尺寸嗎？**  
A：可以。集成材的製程允許拼接出大面積板材，甚至比整塊原木更容易做到特定寬度或長度的訂製。這也是集成材在訂製家具市場中被廣泛採用的原因之一。

**Q5：集成材的表面可以重新打磨保養嗎？**  
A：可以。集成材是實木，表面可以用砂紙打磨、重新上油或上漆，保養方式和全實木家具相同。但若指接縫有膠合劑外滲的痕跡，打磨時要特別注意均勻施力，避免接縫處顏色不一致。
