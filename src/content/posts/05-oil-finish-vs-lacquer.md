## 推油：讓木頭從內部被保護

### 原理

推油（Oil Finish）的邏輯是讓油脂「滲入」木材細胞壁，從內部滋潤並形成保護。木材本身的毛細孔依然開放，木頭依然能呼吸。

常見的推油材料：

**木蠟油（Hard Wax Oil）**  
目前職人圈最常用的配方。結合了天然硬蠟（Carnauba Wax、Candelilla Wax）和植物油（亞麻仁油、葵花油），滲透木纖維後，硬蠟成分在表面形成一層超薄的保護膜，既有油的滋潤感，又有蠟的耐磨性。代表品牌：德國 Osmo（奧斯莫）、英國 Treatex。

**丹麥油（Danish Oil）**  
亞麻仁油、桐油的混合配方，早期北歐木工師傅愛用。滲透性強，保護膜薄，木材感最強烈，但耐水性比木蠟油稍差。

**亞麻仁油（Raw Linseed Oil）**  
最傳統的塗裝用油，純天然，但乾燥時間很長（3-7天），現在多作為其他油品的基底。

**椰子油、芝麻油**  
部分工坊會用食用油系列，優點是無毒安全，適合餐具接觸面；缺點是保護效果較弱，需要更頻繁補護。

### 推油完成的外觀

推油的桌面不會有「亮亮的膜」感。木紋清晰、觸感真實，摸起來就是木頭，而不是在摸塑膠。顏色通常比上漆稍微深一點（油脂讓顏色飽和度提升），但視覺上更「自然」。

如果你追求的是那種「這是一塊真正的木頭」的感覺，推油是你要的答案。

## 上漆：在木頭表面建立護盾

### 原理

上漆（Lacquer / Varnish）的邏輯完全不同——不是滲入木頭，而是在表面形成一層硬膜。木材本身的毛細孔被封閉，木頭不再直接接觸空氣。

常見的塗裝材料：

**PU 漆（Polyurethane Lacquer）**  
台灣最常見的傢俱用漆。耐磨、耐水、好清潔，但質感偏「塑膠感」，特別是高光面。分為水性和油性，水性 PU 漆環保性較好。

**硝基漆（Nitrocellulose Lacquer）**  
老式工藝，表面質感比 PU 更自然，但硬度較低、耐候性較差。現在多用於樂器或古董傢俱修復。

**UV 漆（UV Curing Lacquer）**  
用紫外線快速固化，工廠生產線常用，硬度高、速度快，但現場工坊少用（需要 UV 設備）。

**水性漆（Water-based Finish）**  
近年環保意識抬頭，水性漆的市占率在台灣傢俱業快速提升。揮發性有機物（VOC）低，乾燥後無味，與油性漆的耐用度差距已大幅縮小。

### 上漆完成的外觀

全消光漆：表面有均勻霧面感，比推油的「裸木感」多一點隔離感，但仍算自然。  
半光漆：輕微光澤，最常見。  
全光漆：高亮鏡面感，木紋清晰但有「玻璃蓋在上面」的感覺。

| 比較項目 | 推油（木蠟油） | 上漆（PU漆） |
|---------|-------------|------------|
| 工法原理 | 油脂滲入木纖維 | 硬膜覆蓋木材表面 |
| 觸感 | 裸木感，最真實 | 有膜感，隔了一層 |
| 外觀 | 自然、低光澤 | 可選消光到全光 |
| 耐水性 | 中等（需定期補護） | 高（硬膜保護） |
| 耐磨性 | 中等 | 高（特別是 PU） |
| 耐熱杯墊 | 低（容易留水環痕） | 中等 |
| 維護頻率 | 每半年至一年補油 | 基本免維護（直到剝落） |
| 維護難度 | 容易（自己 DIY） | 困難（需補漆） |
| 修復性 | 高（砂磨後重新推油） | 低（補漆需要技術） |

| 環保性 | 高（植物油基） | 中等（含溶劑） |

| 適合家有幼兒 | 高（無毒配方可選） | 中等（固化後安全） |
| 價格（工藝成本） | 較低 | 較高（多道工序） |

## 觸感體驗：這是最容易被忽略的差異

這是我最想讓你真正感受到的點。

推油的桌子——你把手掌放上去，那個溫潤感是真的木頭的溫度。木纖維的細微粗糙感，帶著油脂的絲滑。就像摸一個人的手背，有溫度、有紋理、有彈性。

上漆的桌子——你把手掌放上去，那個涼感是漆膜的溫度。平滑、均勻、乾淨。就像摸一塊玻璃，或者一個好的手機螢幕。

這不是哪種更好，而是哪種更符合你的生活感受。

如果你買實木桌的原因是「喜歡那個木頭的真實感」——選推油。
如果你買實木桌的原因是「耐用、好清潔、不需要管」——選上漆。

## 台灣氣候下哪種更適合？

台灣的高溫高濕環境對兩種塗裝都有影響，但方式不同。

### 台灣濕氣對推油的影響

推油的木頭毛細孔開放，因此木頭仍然會隨濕度收縮膨脹（這是正常的，也是木頭的優點——它在「呼吸」）。  
問題：台灣梅雨季濕度過高時，推油款桌面可能吸水後顏色局部加深，或者出現白霧感（水分停留在木纖維中）。  
對策：定期補護（至少每年一次），梅雨季保持室內通風，不讓水分長時間停留在桌面。

### 台灣濕氣對上漆的影響

上漆的硬膜封閉了木材表面，濕氣不直接接觸木頭，短期保護性更好。  
問題：台灣高溫＋長期使用下，硬膜容易在接縫或邊角處開始剝離，一旦剝離，水分就會從缺口侵入，這時候修復比推油更困難。  
對策：接縫、桌角是重點觀察位置，一旦發現剝漆要盡快補，不然水分侵入後整面都要重刷。

### 南台灣（台南、高雄）特別建議

台南的特色是：乾季夠乾、濕季夠濕、日曬強烈。

日曬是上漆的大敵——紫外線會讓漆膜加速老化，特別是靠窗的傢俱，你可能發現漆面顏色不均、出現微細裂紋，這是正常的老化現象，但速度比北部更快。

從台南氣候出發，我的建議是：

- **餐桌、客廳傢俱（主要室內）**：推油或消光 PU 漆都可以，推油更好維護。
- **餐桌（有幼兒、常有茶水翻倒的家庭）**：半光 PU 漆耐用性較好，犧牲一點觸感。
- **書房書桌（輕度使用）**：強烈推薦推油，觸感體驗最好，台灣書房通常有空調，濕度控制較好。
- **靠窗傢俱**：推油款，因為推油老化後只需補護，不像上漆老化後會出現難看的剝落感。

| 場景 | 推薦塗裝 | 原因 |
|------|---------|------|

## 推油的 DIY 維護教學

推油款傢俱的一大優勢就是自己可以保養，不需要找師傅。

### 工具準備

- 240 號砂紙（細砂，順紋打磨用）
- 木蠟油一罐（Osmo Polyx-Oil 或同等級）
- 白棉布 2 條（一條上油、一條擦拭）
- 手套

### 步驟

**1. 清潔桌面**  
用濕布擦拭，確認沒有灰塵和污漬，自然風乾。

**2. 輕砂**  
用 240 號砂紙順著木紋方向輕磨桌面。不需要太用力，目的是去除表面老化的油脂和細微汙垢，讓新的油能滲入。砂磨後用乾布擦掉木屑。

**3. 上油**  
用棉布沾少量木蠟油，薄薄塗在桌面——不需要厚塗，油是要滲入的，不是堆在表面的。以圓形或打圈方式均勻推開，讓油脂慢慢滲透。

**4. 靜置滲透**  
等 15-30 分鐘，讓油脂充分滲入木纖維。

**5. 擦拭多餘油脂**  
用乾淨棉布用力擦掉表面多餘的油脂——這步驟很重要，殘留太多油在表面會黏手。

**6. 靜置固化**  
至少 12 小時不使用桌面，讓油脂完全固化。

整個過程約 1 小時，之後你的桌子會明顯感覺到木色變深、質感提升。每年做一次，你的桌子可以用一輩子。

## 常見問題

**Q1：推油的桌子滴上水會留印嗎？**

A：會，特別是新的推油桌面。水分若停留超過 10-15 分鐘，可能在木纖維表面留下淡淡的水環。但這不是永久的——乾燥後通常消退，或者下次補護時一起處理。如果你家有小孩常常打翻飲料，建議選 PU 漆，或者在桌面加玻璃墊。

**Q2：木蠟油是什麼？跟一般油差在哪裡？**

A：木蠟油（Hard Wax Oil）是油和硬蠟的複合配方。一般油（如丹麥油、亞麻仁油）只有滲透保護，表面沒有蠟的硬度；木蠟油在滲透之後，蠟成分在表面形成超薄的保護膜，既保留木頭的自然感，又增加了一定的耐磨性。目前職人圈最推崇的是德國 Osmo，台灣可以在傢俱材料行或網路買到。

**Q3：PU 漆用久了之後可以換成推油嗎？**

A：可以，但過程比較麻煩。需要先用砂紙（120 號開始）把舊漆全部磨掉，露出木材本身，再重新推油。這個工程不小，建議交給有經驗的師傅。直接在舊漆上推油是無效的，油滲不進去。

**Q4：推油傢俱放在廚房旁邊可以嗎？**

A：廚房旁邊的傢俱會接觸到水汽、油煙和高溫，對推油款來說是高挑戰環境。如果你的廚房有強力排煙裝置、且傢俱不會直接被水潑到，推油款加上每半年補護是可以的。如果廚房通風差、常有水氣，建議選 PU 漆或者至少選耐水性較好的木蠟油（而非丹麥油）。

**Q5：Osmo 木蠟油要在哪裡買？多少錢？**

A：台灣大型傢俱材料行（如三重、台中的材料行）有售，線上蝦皮也找得到。Osmo Polyx-Oil 透明款（750ml）約台幣 1,000-1,400 元，可以用很多次（一張餐桌大概用掉 50-100ml）。雖然不便宜，但這是目前市面上評價最好的木蠟油，非常值得。
