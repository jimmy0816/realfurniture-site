## 為什麼二手實木家具值得買？

### 三大優勢

**1. 價格殺很大**

二手實木家具普遍 3-6 折，名廠品牌甚至可以 2-4 折入手。一張詩肯柚木的二手餐桌，新品要價 8-12 萬，二手市場常見 3-5 萬成交。

**2. 甲醛已揮發，比新品更安全**

實木家具（尤其是有木料接合用到黏膠的集成材）的甲醛釋放高峰在前 6-12 個月。買一張用了 2-3 年的二手實木家具，甲醛問題幾乎不存在了。

**3. 更環保**

使用二手家具減少生產需求，降低碳排放。台灣有質感的實木家具值得被好好使用，不應該只因為「想換新風格」就進垃圾場。

### 適合買二手的族群

- 預算有限，但不想妥協材質
- 剛租屋、可能搬家的族群（二手好轉售）
- 喜歡中古/老件質感的人
- 對環境友善有意識的消費者

## PTT 二手傢俱版：老派但有寶

PTT 的 **Furniture 版**（家具板）和 **flea_market（跳蚤市場）** 一直是台灣二手家具的老牌管道，使用者以 30-50 歲居多，發文多為誠信賣家，但流通量近年不如 FB 和 Carousell 活躍。

### 怎麼找？

1. 進入 PTT（mobile01 也有二手傢俱版）
2. 搜尋關鍵字：「實木」「胡桃木」「柚木」「餐桌」
3. 注意發文日期，超過 3 個月的多半已售出

### PTT 找二手實木的技巧

- **主題搜尋**：在 PTT Furniture 版搜尋樹種名稱（胡桃木、橡木、柚木）
- **跳蚤市場（flea_market）**：家具類別，可找到散戶賣家
- **問版友推薦**：PTT 版友通常對品牌和行情很了解，發文問「XX 預算找實木餐桌建議」常有意外收穫

### 注意事項

PTT 沒有交易保護機制，建議僅限面交，不接受貨到付款以外的轉帳模式。

## Yahoo 拍賣：流通量最大的選擇

**Yahoo 拍賣（tw.bid.yahoo.com）** 是台灣二手家具流通量最大的單一平台，從普通家具行清貨、個人搬家求售，到專業二手家具商都在上面。

### 如何精準搜尋實木家具

搜尋關鍵字組合建議：
- `二手 實木餐桌`
- `中古 胡桃木桌`
- `詩肯柚木 二手`（詩肯是知名柚木品牌，二手保值）
- `實木 餐桌椅組 七成新`

### 篩選技巧

1. **按「最新」排序**：好貨通常很快被買走
2. **設定地區**：選你所在縣市，省去外縣市搬運費（大型家具搬運費可能和家具本身一樣貴）
3. **確認賣家評價**：查看賣家歷史評價，避免詐騙
4. **看多張圖**：要求賣家提供桌面特寫、桌腳接合處、側面剖面照

### Yahoo 拍賣上的知名二手家具商

以下是 Yahoo 拍賣上有固定店面的二手家具業者（以 2026 年搜尋結果為準）：

| 業者名稱 | 主要地區 | 特色 |
|---------|---------|------|
| **宏品二手家具** | 台北/台中 | 老字號，品項多，有實體店可看貨 |
| **永茂二手家具館** | 台中 | 原木家具比例高 |
| **百豐悅** | 台北 | 品牌二手傢俱，DREXEL HERITAGE 等精品 |
| **樂居二手家具** | 新竹/台北 | 價格親民，IKEA、Nitori 為主 |

> ⚠️ 業者資訊以您搜尋時的最新狀態為準，建議購買前確認電話和地址。

## Carousell 旋轉拍賣：新手友善

**Carousell（tw.carousell.com）** 是近年崛起的 C2C 二手平台，介面比 Yahoo 拍賣現代，賣家多為個人，商品照片通常拍得比較好（因為用戶多年輕世代）。

### 旋轉拍賣的優勢

- **APP 介面友善**：手機操作比 Yahoo 拍賣順暢
- **賣家多為個人**：通常有完整的使用背景（買了多久、為何出售）
- **即時通訊**：可直接在 APP 內聊天議價
- **照片品質較好**：年輕賣家通常拍照有審美意識

### 搜尋關鍵字建議

- `實木餐桌`
- `胡桃木 家具`
- `柚木 桌`
- `原木 餐桌`

### 注意事項

旋轉拍賣沒有強制交易保護，仍建議面交或要求到付。

## Facebook 社團：最即時的好貨

**Facebook 社團**是目前台灣二手家具流通最快的管道——好東西發文後幾小時就被搶走，需要勤看、快反應。

### 推薦搜尋的 FB 社團名稱

在 Facebook 搜尋以下關鍵字可找到相關社團：

| 搜尋關鍵字 | 說明 |
|---------|------|
| `台灣二手家具買賣` | 全台通用型社團，人數多 |
| `二手傢俱買賣交流` | 常見社團類型 |
| `台北二手家具` | 北部地區限定 |
| `台中二手家具` | 中部地區限定 |
| `高雄二手家具` | 南部地區限定 |
| `IKEA 二手交流` | 專找 IKEA 家具 |
| `詩肯 二手` | 找詩肯柚木品牌 |

> **注意**：FB 社團名稱隨時可能改變或新社團出現，建議直接在 Facebook 搜尋並加入最活躍的社團（以成員數和近期發文頻率判斷）。

### FB 社團操作技巧

1. **開啟通知**：加入社團後開啟「全部通知」，第一時間看到新貼文
2. **善用關鍵字搜尋**：在社團內搜尋「實木」、「胡桃木」等可快速找到歷史貼文
3. **發文求購**：可主動發「徵求：4-6人實木餐桌，台北，預算 NT$10,000-20,000」，讓賣家主動來找你
4. **查看賣家歷史**：點進賣家個人頁，確認是真實帳號

### 辨別假貨或詐騙 FB 貼文

🚩 以下為警示訊號：
- 照片是網路圖，背景明顯不是台灣家庭環境
- 只有一張圖，拒絕提供更多照片
- 要求先轉帳才約看貨
- 價格異常便宜（如全新外觀胡桃木餐桌只賣 3,000 元）

## 議價與看貨技巧

### 議價策略

1. **先查新品售價**：知道市場行情，才能判斷二手的合理價位
2. **一般合理折扣**：7-8 成新的家具約新品 4-6 折是合理範圍
3. **議價切入點**：提出具體理由（「我看到這邊有刮痕，可以算 XX 嗎？」）比直接殺價有效
4. **搭配購買**：想買多件時，整批議價通常更有彈性
5. **現金付款**：部分賣家對現金有額外折扣

### 看貨重點

看實木二手家具時必查：

| 檢查項目 | 方法 | 問題訊號 |
|---------|------|---------|
| **桌面平整度** | 手掌平放滑過 | 有凹凸波紋表示有翹曲 |
| **桌腳穩定度** | 輕搖桌腳 | 有明顯晃動表示榫卯已鬆 |
| **木材是否開裂** | 用手電筒照射 | 有裂縫特別是貫穿裂需注意 |
| **貼皮完整度** | 看邊角和桌緣 | 起翹代表是貼皮家具且保養不佳 |
| **異味** | 靠近聞 | 強烈霉味或刺鼻化學味是警訊 |
| **螺絲孔磨損** | 看接合處 | 過大或重複鎖過的螺絲孔會影響結構 |

## 避坑清單：二手買家必知

- [ ] **確認是全實木還是貼皮**：看側面剖面和邊角
- [ ] **詢問使用年限和保養歷史**：3 年內使用過木蠟油的家具更值得買
- [ ] **確認尺寸符合你家空間**：大型家具搬入搬出費用不少
- [ ] **計算搬運費**：外縣市搬運費可能比家具本身貴
- [ ] **不接受只看「樣品照」**：要求現況照（室內、自然光）
- [ ] **避免只轉帳不見面**：除非賣家有良好評價記錄
- [ ] **詢問是否有原廠保固轉移**：部分台灣品牌保固可隨家具轉移

## FAQ

**Q：二手實木家具買回來要怎麼處理？**

先徹底清潔（用白醋水稀釋液擦拭），完全乾燥後重新上一層木蠟油。這樣可以去除前主人的氣味，同時補充木材油脂，讓家具煥然一新。

**Q：Yahoo 拍賣和 Carousell 哪個好用？**

Yahoo 拍賣商品量更多（包含業者），Carousell 個人賣家較多、照片較好。建議兩個都看，找到目標商品類型後在較活躍的平台深挖。

**Q：有沒有二手實木家具的「合理價格」參考？**

大致參考：
- 7-9 成新品牌實木餐桌（如詩肯）：新品 4-6 折
- 5-7 成新台灣一般品牌實木桌：新品 2-4 折  
- 原廠不可考、狀況不詳：新品 1-3 折
- IKEA 實木二手：新品 3-5 折（流通量大，行情透明）

**Q：台灣二手家具用什麼 APP 找最方便？**

- **Carousell 旋轉拍賣**（APP）：介面最好用
- **Facebook**（APP）：社團功能，需要主動搜尋和加入
- **Yahoo 拍賣**（APP/網頁）：商品量最大
- **PTT**（APP/網頁）：進階玩家、特定品牌玩家用
