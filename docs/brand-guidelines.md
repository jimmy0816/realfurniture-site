# 里爾家具 RealFurniture — 品牌規範 v1.0

> 最後更新：2026-03-10

---

## 品牌核心

**品牌名稱**：里爾家具研究室 ｜ RealFurniture
**Tagline**：「每一件家具，都是一次與木的對話」
**英文 Tagline**：Every piece of furniture is a conversation with wood.

**品牌定位**：台灣高級木製傢俱的深度研究基地
- SEO 內容站，以專業評測 + 工藝知識建立搜尋權威
- 為「里爾家具」（台南永康，FB: RealFurnitureTW）導流

## 目標受眾

**主要**：30-55 歲台灣中產，正在裝潢或換家具
- 搜尋「實木餐桌推薦」「客製家具台南」的人
- 願意為品質付費，但怕被貼皮騙
- 從 Google 搜尋進站

**次要**：設計師、室內裝修業者
- 搜尋木材知識、工法比較
- 可能轉介紹客戶

## 語氣規則

### ✅ Do
- 用「我們實測」「師傅建議」等有根據的說法
- 提供具體數據：含水率、承重、尺寸
- 用比較表讓讀者快速決策
- 每篇結尾導向 FB 聯繫

### ❌ Don't
- 不說「最好的」「第一名」（除非有數據支撐）
- 不貶低其他品牌
- 不用過度行銷語言（「驚爆價」「限時搶購」）
- 不捏造評測數據

### 設計原則
- **留白為王**：讓木製品攝影成為焦點
- **方正按鈕**：border-radius: 0（工藝的精確感）
- **Serif + Sans 搭配**：傳遞「工藝 × 現代」
- **行動版優先**：受眾從 FB 行動端進站

## 攝影風格

| 類型 | 規格 |
|------|------|
| 產品攝影 | 自然光 / 側光 / 暖色溫 4500-5500K / 淺景深突出木紋 |
| 工藝過程 | 紀實風格 / 手部特寫 / 工具使用 |
| 背景 | 淺灰牆面或自然環境，不用純白背景 |
| 濾鏡 | 可加微量 film grain，不過度調色 |
| 禁止 | Stock photo、AI 生成圖、過度 HDR |

## 轉換策略

**目標**：Google 搜尋 → 閱讀文章 → 點擊 CTA → FB 粉專聯繫

| 觸點 | CTA 位置 | 追蹤 |
|------|---------|------|
| 導航列 | 右上「聯繫我們」 | `utm_content=nav` |
| Hero | 「聯繫我們」按鈕 | `utm_content=hero` |
| 文章側邊欄 | Sticky「聯繫我們」 | `utm_content=sidebar` |
| 文章底部 Banner | 「聯繫我們」 | `utm_content=banner` |
| 分類頁側邊欄 | 「聯繫我們」 | `utm_content=category-sidebar` |

所有 CTA 導向：`https://www.facebook.com/RealFurnitureTW?utm_source=realfurniture.tw&utm_medium=cta&utm_campaign=contact`

**GA4**：`G-EMEKLFZ4CR`，事件 `cta_click` 追蹤所有按鈕點擊
